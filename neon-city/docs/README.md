# Neon City

A dark coding color scheme using a **Midnight** base with an **Electric Neon Spectrum** — magenta functions, plasma green strings, cyber blue types, neon orange numbers, hot pink constants, and **Bold White** keyword anchors. Built for the night shift — maximum contrast, zero fatigue.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0D0221` | Midnight |
| Sidebar / Panels | `#07010F` | Void Black |
| Keywords | `#FFFFFF` | Bold White |
| Functions | `#FF2CF1` | Electric Magenta |
| Strings | `#39FF14` | Plasma Green |
| Types & Tags | `#00D4FF` | Cyber Blue |
| Numbers | `#FF6B2B` | Neon Orange |
| Constants & Booleans | `#FF69B4` | Hot Pink |
| Variables | `#E0D4FF` | Neon White |
| Comments | `#6B5B95` | Twilight Indigo |
| Attributes | `#FFE744` | Neon Yellow |
| Errors | `#FF1744` | Alarm Red |

## Key Ideas

- **Electric neon spectrum.** Each syntax role gets its own vivid neon color — no two roles share a color. Maximum differentiation, maximum clarity.
- **Magenta hero.** Electric Magenta remains the dominant accent for functions and methods — the neon sign that draws your eye.
- **White keyword anchors.** Bold white keywords mark structural flow — the skeleton you scan first.
- **Depth over borders.** Panels separated by shade, not lines.
- **Cyan cursor.** Electric cyan — always visible, always unmissable.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
