# Thermal Shelf

A dark coding color scheme using a **deep stratospheric base** with an **altitude spectrum** — teal functions, golden strings, cloud-blue types, warm amber numbers, ivory constants. Built for the stillness at the edge of atmosphere.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0F1B24` | Stratosphere Dark |
| Sidebar / Panels | `#091318` | Deep Altitude |
| Keywords | `#F0E6D0` | Cream White |
| Functions | `#4EC9B0` | Thermal Teal |
| Strings | `#D4A856` | Cloud Gold |
| Types & Tags | `#A8D8EA` | Cirrus Blue |
| Numbers | `#E8943A` | Warm Amber |
| Constants & Booleans | `#E8DCC8` | Ivory |
| Variables | `#D8E4E8` | Frost White |
| Comments | `#6B8FA0` | Altitude Slate |
| Attributes | `#E8DCC8` | Ivory |
| Errors | `#E05252` | Coral Red |

## Key Ideas

- **Altitude spectrum.** Colors drawn from the boundary where warm thermals meet cold stratosphere — teal, gold, blue, amber.
- **Teal hero.** Functions blaze in thermal teal (`#4EC9B0`) — the dominant color of the atmosphere at altitude.
- **Cream keywords.** Bold ivory-cream anchors mark structural flow without competing with the spectrum.
- **Depth over borders.** Panels separated by shade — deep altitude against stratosphere dark.
- **Amber cursor.** The warm amber caret (`#E8943A`) is sunlight catching a cloud edge — unmissable against the cool base.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
