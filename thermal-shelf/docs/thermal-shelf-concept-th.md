# Thermal Shelf — แนวคิดชุดสี

## ที่มา

ภาพต้นแบบเป็นนักบินอวกาศยืนอยู่บนขอบหน้าผาเหนือทะเลเมฆขนาดใหญ่ เมฆถูกแสงแดดส่องจากด้านหลัง — ผิวเมฆเป็นสีครีมทอง (`#D4A856`) ส่วนเงาจมลงเป็นสีเทาอมฟ้า ท้องฟ้าด้านบนไล่จากสีเขียวอ่อนที่ขอบฟ้าไปจนถึงสีน้ำเงินเข้มของชั้นสตราโตสเฟียร์ (`#0F1B24`) ที่จุดสูงสุด ฉากทั้งหมดอยู่ที่จุดแบ่งทางความร้อน (thermal boundary) — ระดับความสูงที่อากาศร้อนที่ลอยขึ้นมาพบกับชั้นบรรยากาศเย็นด้านบน ทำให้เกิดเมฆคิวมูลัสขนาดยักษ์

---

## วิสัยทัศน์

Thermal Shelf เป็นชุดสีสำหรับเขียนโค้ดที่ได้แรงบันดาลใจจากช่วงเวลาก่อนที่ชั้นบรรยากาศจะสิ้นสุด แรงตึงหลักของธีมคือเรื่อง "ความร้อน" — แสงสีทองอุ่นดันขึ้นผ่านเมฆ ขณะที่ความเย็นสีน้ำเงินดำของอวกาศกดลงมา

ธีมนี้แก้ปัญหาแรงตึงดังกล่าวผ่านการออกแบบ 3 จุด:

- **พื้นหลังสตราโตสเฟียร์** — สีพื้น (`#0F1B24`) ไม่ใช่สีดำเปล่าๆ แต่เป็นสีน้ำเงินอมเขียวเข้มเฉพาะของท้องฟ้าที่ระดับ 40,000 ฟุต มีความเย็นของความสูงแต่ไม่ว่างเปล่าเหมือนอวกาศ
- **สีอุ่นจากชั้นเมฆ** — Gold (`#D4A856`), Amber (`#E8943A`), Ivory (`#E8DCC8`) มาจากผิวเมฆที่ถูกแสงแดด ใช้กับ strings, numbers และ constants — ข้อมูลที่เป็นรูปธรรม
- **สีเย็นจากชั้นบรรยากาศบน** — Teal (`#4EC9B0`), Cirrus Blue (`#A8D8EA`), Cyan (`#56C8D8`) มาจากท้องฟ้าและแสงกระจายในเงาเมฆ ใช้กับ functions, types — โครงสร้างของโค้ด

ผลลัพธ์คือพื้นที่ทำงานที่ตาแยกส่วน "ข้อมูล" (สีอุ่น) ออกจาก "โครงสร้าง" (สีเย็น) ได้อย่างเป็นธรรมชาติ

---

## หลักการออกแบบ

### 1. เส้นแบ่งทางความร้อน — ข้อมูลสีอุ่น, โครงสร้างสีเย็น

สีเน้นทุกสีในชุดนี้มาจากฝั่งใดฝั่งหนึ่งของเส้นแบ่งอุณหภูมิในภาพ Functions (`#4EC9B0`) เป็นสี Teal เพราะเป็นโครงสร้าง — กำหนดสถาปัตยกรรมของการทำงาน เหมือนชั้นบรรยากาศเย็นที่กำหนดเพดานของเมฆ Strings (`#D4A856`) เป็นสี Gold เพราะเป็นข้อมูล — ค่าที่มีความหมายจริงๆ เหมือนอากาศร้อนที่พาความชื้นขึ้นไปบนฟ้า Types เป็นสี Cirrus Blue (`#A8D8EA`) — เบากว่า กระจายกว่า เหมือนเมฆบางๆ ที่ก่อตัวที่จุดสูงสุดของเสาอากาศร้อน Numbers เป็นสี Amber (`#E8943A`) — จุดที่แสงแดดร้อนที่สุดทะลุผ่านช่องว่างของเมฆ

### 2. โครงสร้างเงียบ, เนื้อหาสว่าง

Keywords เป็นสี Cream White (`#F0E6D0`) ตัวหนา — ทำเครื่องหมายโครงกระดูกของโค้ด (if/else, return, function) โดยไม่แย่งความสนใจจากเนื้อหา เหมือนพื้นที่นักบินอวกาศยืนอยู่: จำเป็น มั่นคง แต่ไม่ใช่ตัวเอกของภาพ ลำดับความสำคัญทางสายตา:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Keywords | Cream White | `#F0E6D0` | แท่นยืน — โครงสร้าง ตัวหนา สีกลางอุ่น |
| Functions | Thermal Teal | `#4EC9B0` | ตัวเอก — สีบรรยากาศหลักที่ระดับความสูง |
| Strings | Cloud Gold | `#D4A856` | ผิวเมฆที่ถูกแสง — อุ่น เป็นข้อมูลจริง |
| Types | Cirrus Blue | `#A8D8EA` | เมฆน้ำแข็งชั้นสูง — โครงสร้างแต่เบากว่า functions |
| Numbers | Warm Amber | `#E8943A` | แสงที่ร้อนที่สุด — เฉพาะเจาะจง สว่าง เข้มข้น |
| Constants | Ivory | `#E8DCC8` | สีขาวเมฆ — ค่าคงที่ มีตัวตนแต่ไม่เปลี่ยนแปลง |
| Comments | Altitude Slate | `#6B8FA0` | หมอกระหว่างชั้นเมฆ — มองเห็นได้แต่ถอยลง |
| Variables | Frost White | `#D8E4E8` | กระจกหน้าหมวก — เป็นกลาง ใส สะท้อน |

### 3. พื้นหลังเป็นเวที ไม่ใช่ความว่างเปล่า

พื้นหลัง `#0F1B24` เป็นสีเข้มอมน้ำเงินที่อ่านเป็น "ความลึก" ไม่ใช่ "ความว่าง" เป็นสีของท้องฟ้าเหนือชั้นเมฆพอดี: ไม่ดำสนิทเหมือนอวกาศ ไม่ดำเหมือนจอปิด แต่เป็นความมืดที่ยังมีบรรยากาศ รายละเอียด:

- **พื้นผิว editor** (`#0F1B24`) — ชั้นสตราโตสเฟียร์เหนือเมฆ มืดพอสำหรับใช้งานนานๆ มีสีน้ำเงินพอให้รู้สึกถึงความสูง
- **Panel/sidebar** (`#091318`) — ลึกกว่า editor เหมือนมองลงไปในช่องว่างระหว่างเมฆที่มองไม่เห็นพื้นดิน
- **Highlight** (`#1A2D3A`) — ความสว่างที่เพิ่มขึ้นเล็กน้อยเมื่อเมฆรับแสงจากด้านข้าง การเลือกข้อความยกตัวขึ้นเบาๆ
- **Errors** ใช้สี Coral Red (`#E05252`) — อุ่นพอที่จะเข้ากับชุดสี แตกต่างจากกลุ่ม amber/gold พอที่จะบอกปัญหาได้โดยไม่ทำให้ตกใจ

### 4. เคอร์เซอร์เป็นจุดยึด

เคอร์เซอร์ใช้สี Warm Amber (`#E8943A`) ในภาพ จุดที่สว่างที่สุดคือตรงที่แสงแดดตรงทะลุช่องว่างในกำแพงเมฆแล้วไปโดนขอบของหอเมฆ — เส้นแสงสีทองเข้มข้นเส้นเดียวตัดกับสีฟ้าเทาโดยรอบ เคอร์เซอร์สี Amber จำลองผลนี้ตรงๆ: จุดแสงอุ่นคมชัดบนพื้นหลังเย็น ตาจะกลับมาหาจุดนี้เสมอ เหมือนตาที่กลับไปหาจุดสว่างที่สุดในทิวทัศน์

## การจับคู่แบบอักษร

ชุดสี "Thermal Shelf" เข้าคู่กับ:

- **หลัก:** JetBrains Mono — ความแม่นยำแบบเครื่องจักรเข้ากับสภาพแวดล้อมทางวิศวกรรมของนักบินอวกาศ ความหนาเส้นสม่ำเสมอทำให้สีเน้นอุ่น/เย็นอ่านได้ชัดที่ขนาดเล็กบนพื้นหลังเข้ม
- **ทางเลือก:** Fira Code — ปลายตัวอักษรโค้งมนกว่าเล็กน้อย ทำให้สี Gold และ Amber หายใจได้มากขึ้น

Ligatures อย่าง `=>`, `!==` และ `>=` เสริมความรู้สึกของระบบวิศวกรรม — อ่านเหมือนตัวอักษรบนแผงควบคุมบนกระเป๋าสะพายของนักบินอวกาศ

## ความรู้สึกในการใช้งานจริง

ลองจินตนาการเปิดไฟล์ TypeScript ตอนตี 2 ห้องมืด หูฟังเปิดเสียงต่ำๆ พื้น editor ไม่ใช่สีดำ — เป็นสีน้ำเงินอมเขียวเข้มของท้องฟ้าที่ยังมีบรรยากาศอยู่ ยังมีความลึก ตาเคลื่อนไปที่ functions สี Teal ก่อน: สีที่ดังที่สุด โครงกระดูกของโครงสร้าง ดึงคุณเข้าไปในโค้ดเหมือนเส้นขอบฟ้าดึงตาในภาพถ่าย ระหว่าง functions strings สี Gold จับแสง — อุ่น หนักแน่น พาข้อมูลจริงของโปรแกรม Types ลอยอยู่ในสี Cirrus Blue เบาและกระจาย บอกขอบเขตของสิ่งต่างๆ โดยไม่แข่งกับสิ่งที่สิ่งต่างๆ ทำ Numbers เผาไหม้ในสี Amber: เฉพาะเจาะจง ร้อน เข้มข้น เนื้อหาที่สว่างที่สุดบนหน้าจอ Keywords สี Cream White ยึดบรรทัดไว้ด้วยกันอย่างเงียบๆ — หนาแต่ไม่มีสี เป็นโครงสร้างแต่ไม่ใช่การตกแต่ง Comments ถอยลงไปในสี Altitude Slate มีตัวตนแต่เป็นบรรยากาศ อ่านได้เมื่อตั้งใจมอง และตรงกลางของทั้งหมด เคอร์เซอร์สี Amber กะพริบ: จุดอุ่นจุดเดียวบนทุ่งสีเย็น สีเดียวกับแสงแดดทะลุกำแพงเมฆ ตาของคุณรู้เสมอว่าอยู่ตรงไหน

นั่นคือ thermal-shelf.
