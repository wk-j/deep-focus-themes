# Silica Haze — แนวคิดชุดสี

## ที่มา

ธีมนี้มาจากภาพทิวทัศน์แนว science fiction ที่แสดงเนินทรายกว้างใหญ่ภายใต้ชั้นบรรยากาศที่เต็มไปด้วยฝุ่นทราย บนดาวเคราะห์ที่มีดวงจันทร์สองดวง อากาศไม่โปร่ง แต่เต็มไปด้วยผงซิลิกาลอยอยู่ ทำให้แสงที่ผ่านเข้ามาเป็นสีอุ่น กระจายตัว และไม่มีทิศทางชัดเจน ดวงจันทร์เสี้ยวสองดวงลอยอยู่ในส่วนบนของภาพ เป็นสิ่งเดียวที่มีขอบคมชัด ส่วนที่เหลือทั้งหมดไล่เฉดจากสีครีม ทอง น้ำตาลแดง ไปจนถึงน้ำตาลเข้มเกือบดำ

---

## แนวคิดหลัก

silica-haze เป็นชุดสีสำหรับเขียนโค้ดที่มาจากโลกที่ชั้นบรรยากาศทำจากวัสดุเดียวกับพื้นดิน ปัญหาหลักในการออกแบบคือ ทุกอย่างในภาพอยู่ในโทนอุ่นเดียวกันหมด แต่เราต้องแยกแยะ token ต่างๆ ในโค้ดให้ได้

ธีมนี้แก้ปัญหานั้นด้วยสามแนวทาง:

- **โทนอุ่นเดียวกันทั้งหมด** — พื้นหลัง (`#1C1410`, `#140E0A`) และตัวอักษร (`#E0D4C0`, `#B0A088`) อยู่ในตระกูลสีน้ำตาล-ครีมเหมือนกัน ทำให้ทั้ง workspace รู้สึกเป็นเนื้อเดียวกัน ไม่ใช่สีที่ถูกประกอบมาจากหลายที่
- **สีเย็นเพียงจุดเดียว** — Dust Teal (`#6AAAA0`) เป็นสีเย็นตัวเดียวในชุด ใช้สำหรับ type และ annotation ที่อธิบายโครงสร้าง ความเย็นของมันทำให้แยกออกจากสีอุ่นรอบข้างได้ทันที
- **ความสว่างเป็นตัวจัดลำดับ** — ในภาพ สิ่งที่สว่างที่สุดคือดวงจันทร์เสี้ยวและสันเนินทราย ในโค้ด token ที่สว่างที่สุดคือ keyword (Pale Ivory `#EAE0CA`) และฟังก์ชัน (Cream Gold `#D4B870`) ยิ่งสว่างยิ่งสำคัญเชิงโครงสร้าง

---

## หลักการออกแบบ

### 1. ชุดสีที่เผาจากเตาเดียวกัน

สีเน้นทุกตัวยกเว้น Dust Teal มาจากฝั่งอุ่นของสเปกตรัม:

- **Cream Gold (`#D4B870`)** — สีของเม็ดทรายที่โดนแสง ใช้สำหรับฟังก์ชัน
- **Copper (`#C8804A`)** — สีด้านเงาของเนินทราย ใช้สำหรับ attribute และ literal
- **Dune Sienna (`#A06838`)** — สีกรวดหยาบที่พื้นหน้า ใช้สำหรับ constant
- **Spore Green (`#8AAA68`)** — สีเขียวอมเหลืองเหมือนไลเคน ใช้สำหรับ string เป็นโน้ตอินทรีย์ตัวเดียวในชุดแร่ธาตุ

Dust Teal ถูกเลือกแทนสีน้ำเงิน เพราะน้ำเงินจะดูเหมือนท้องฟ้า ซึ่งไม่มีให้เห็นในภาพนี้ Teal มีเขียวผสมพอที่จะอ่านเป็นแร่ธาตุ ไม่ใช่ชั้นบรรยากาศ

### 2. ความสว่างตามน้ำหนักเชิงโครงสร้าง

ลำดับของ token ถูกจัดตามความสว่าง เหมือนกับที่สิ่งสว่างในภาพมีขอบเขตชัดเจนกว่าสิ่งมืด:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Keyword | Pale Ivory | `#EAE0CA` | ดวงจันทร์เสี้ยว — สว่างที่สุด ขอบเขตชัดที่สุดในหมอก |
| ฟังก์ชัน | Cream Gold | `#D4B870` | สันเนินทรายที่โดนแสง — พื้นผิวที่ตาสังเกตเห็นมากที่สุด |
| String | Spore Green | `#8AAA68` | โน้ตอินทรีย์ตัวเดียว เนื้อหาที่มนุษย์เขียนให้มนุษย์อ่าน |
| Type | Dust Teal | `#6AAAA0` | สีเย็นตัวเดียว สำหรับ annotation เชิงโครงสร้าง |
| Attribute / Literal | Copper | `#C8804A` | ด้านเงาของเนิน ค่าที่เป็นรูปธรรม |
| Constant | Dune Sienna | `#A06838` | กรวดหน้าดิน หนัก มืด ตรึงอยู่กับที่ |
| Comment | Buried Grain | `#7A6850` | ทรายในเงาของตัวเอง มองเห็นได้แต่กำลังจมลง |
| Error | Kiln Red | `#C05040` | ดินเผาที่แตกร้าว สัญญาณปัญหาในวัสดุ |

### 3. พื้นหลังที่มีอุณหภูมิ ไม่ใช่ความว่างเปล่า

พื้นหลังทุกระดับมีโทน umber อุ่นๆ ไม่มีสีน้ำเงิน ไม่มีสีเทากลาง ความมืดในภาพไม่ใช่ความมืดของอวกาศ แต่เป็นความมืดของทรายในเงา ซึ่งยังคงอุ่นอยู่เสมอ

- **พื้นที่แก้ไข** (`#1C1410`) — สีของเงาเนินทรายตรงฐาน ดูดซับแสงส่วนใหญ่แต่ยังอ่านเป็นวัสดุ ไม่ใช่ความว่าง
- **แถบด้านข้าง** (`#140E0A`) — มืดกว่าอีก เหมือนร่องที่ลมแกะสลักระหว่างเนินทราย ไม่มีแสงตรงเข้าถึง
- **Highlight / Selection** (`#2C2018`) — เหมือนสันเนินที่จับมุมแสงได้สว่างกว่าเล็กน้อย ยกเนื้อหาขึ้นมาจากพื้น
- **Error** ใช้ Kiln Red (`#C05040`) — สีดินเผาที่ถูกเผานานเกินจนแตกร้าว ยังเป็นสีดินเผาเหมือนเดิม แต่สื่อถึงความเสียหายในวัสดุ

### 4. เคอร์เซอร์เป็นจุดยึด

เคอร์เซอร์ใช้ Cream Gold (`#D4B870`) สีนี้คือแนวสันเนินทรายที่จับแสงได้มากที่สุด เป็นเส้นบางๆ ที่สว่างที่สุดในทุ่งสีอุ่น ในชุดสีที่เกือบจะเป็น monochrome อุ่นทั้งหมด ความแตกต่างของความสว่างคือสัญญาณที่ชัดที่สุด ตาหาเคอร์เซอร์ได้ทันทีเพราะมันสว่างกว่าทุกอย่างรอบข้าง

## ฟอนต์ที่เหมาะสม

ชุดสี silica-haze ใช้ได้ดีกับ:

- **หลัก:** JetBrains Mono — เส้นสม่ำเสมอให้ความรู้สึกเหมือนตัวอักษรที่แกะสลักลงบนหิน ความเป็นระเบียบของฟอนต์ช่วยสร้างจุดยึดเชิงเรขาคณิตท่ามกลางไล่เฉดสีอุ่นที่นิ่มๆ ของธีม
- **ทางเลือก:** Fira Code — ปลายเส้นโค้งกว่าเล็กน้อย ให้ความรู้สึกเหมือนตัวอักษรที่ประทับลงบนดินเหนียวแล้วขอบอ่อนลงตามเวลา

Ligature เช่น `=>`, `!==`, `>=` ให้ความรู้สึกเหมือนกลไกโบราณ ตัวดำเนินการที่ดูถูกแกะมากกว่าพิมพ์ เชื่อมต่อ token เหมือนร่องที่ลมแกะสลักเชื่อมหน้าเนินทรายเข้าด้วยกัน

## ความรู้สึกเมื่อใช้งานจริง

เปิดไฟล์ Rust ขึ้นมาตอนบ่ายสอง มู่ลี่ปิดครึ่งหนึ่ง ห้องอุ่นอยู่แล้ว พื้นหลัง editor ไม่ใช่สีดำ แต่เป็นสี umber อุ่นๆ เหมือนทรายที่อยู่ในเงามานานแต่ยังไม่เย็นลง แถบด้านข้างมืดกว่า ตาจับเป็นความลึกมากกว่าขอบ เมื่อสแกนไฟล์ keyword ปรากฏเป็นสีงาช้างอ่อน สว่างที่สุด เหมือนดวงจันทร์เสี้ยวในท้องฟ้าฝุ่น ชื่อฟังก์ชันเรืองแสงครีมทองอยู่ข้างใต้ ตาไล่จังหวะของมันเหมือนไล่สันเนินทราย string เป็นสีเขียวอมเหลือง เนื้อหาที่มนุษย์เขียนให้อ่าน type annotation นั่งเงียบๆ ในสี teal เป็นจุดเย็นเดียวในทุ่งอุ่น ตัวเลขและ attribute เข้มขึ้นเป็นสีทองแดง comment จมอยู่ในสีน้ำตาลอ่อน เห็นได้เมื่อตั้งใจมอง และเคอร์เซอร์ครีมทอง เส้นบางๆ ที่สว่างที่สุดในจอ หาเจอได้ทันทีเพราะมันคือจุดที่สว่างกว่าทุกอย่าง

นั่นคือ silica-haze.
