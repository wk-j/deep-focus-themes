# Silica Haze

A dark coding color scheme using a **warm umber base** with a **sand-to-teal desert spectrum** — cream-gold functions, spore-green strings, dusty-teal types, and copper literals. Built for unhurried sessions in dry heat.

## Key Ideas

- **Sand-to-teal spectrum.** The accents move from warm gold through copper to a single cool counterpoint in dusty teal, mirroring the temperature range from sunlit dune crest to shadowed trough.
- **Cream-gold functions.** The hero color carries the luminosity of sand catching haze-filtered light — dominant but never harsh.
- **Pale ivory keywords.** Structural tokens are the near-white of crescent moons seen through atmosphere, the lightest objects in the frame.
- **Depth over borders.** Panels separated by umber shade, not lines.
- **Cream-gold cursor.** The hottest point on a dune ridge — unmissable in a warm field.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
