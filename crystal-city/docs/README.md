# Crystal City

A dark coding color scheme using an **Obsidian** base with a **Crystalline Spectrum** — teal functions, mint strings, glacier types, gold numbers, rose constants, and **Bold White** keyword anchors. Built for clarity in the dark — differentiated syntax, zero fatigue.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0A0E1A` | Obsidian |
| Sidebar / Panels | `#060910` | Deep Void |
| Keywords | `#FFFFFF` | Bold White |
| Functions | `#56D4C8` | Crystal Teal |
| Strings | `#89DDCA` | Mint Glass |
| Types & Tags | `#7EC8E3` | Glacier Blue |
| Numbers | `#F0C674` | City Gold |
| Constants & Booleans | `#E8A0BF` | Dusty Rose |
| Variables | `#D8DEE9` | Frost White |
| Comments | `#5A8A7A` | Sage Teal |
| Attributes | `#B48EF0` | Crystal Violet |
| Errors | `#E06C75` | Signal Red |

## Key Ideas

- **Crystalline spectrum.** Each syntax role gets its own color from a harmonized cool-toned palette — no two roles share a color.
- **Teal hero.** Crystal Teal remains the dominant accent for functions and methods — the most frequent semantic token.
- **White keyword anchors.** Bold white keywords mark structural flow — the skeleton you scan first.
- **Depth over borders.** Panels separated by shade, not lines.
- **Rose cursor.** Always visible, never lost.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
