# Suspended Marl

A dark coding color scheme using a **charcoal coastal base** with a **cool-to-warm mist spectrum** — fog-cyan functions, olive-green strings, stone-mauve types, amber constants, and rust-ochre literals. Built for focused sessions in still, grey air.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#181B1F` | Cliff Dark |
| Sidebar / Panels | `#111417` | Sea Cave |
| Keywords | `#C8D8DC` | Coastal White |
| Functions | `#7DC4D4` | Fog Cyan |
| Strings | `#8AAA60` | Cliff Grass |
| Types & Tags | `#9A8CB0` | Stone Mauve |
| Numbers | `#C4A050` | Amber Dry |
| Constants & Booleans | `#C4A050` | Amber Dry |
| Variables | `#C8D8DC` | Coastal White |
| Comments | `#4A5060` | Figure Dim |
| Attributes | `#C4A050` | Amber Dry |
| Errors | `#C45860` | Marl Red |

## Key Ideas

- **Mist as spectrum.** The palette moves from cold fog-cyan through neutral stone to warm rust-ochre, mirroring the temperature gradient of coastal atmosphere at distance.
- **Fog-cyan functions.** The most dominant token takes the color of diffused light through sea-mist — distinct, luminous, never harsh.
- **Near-white keywords.** Structural syntax carries the pale chalk tone of cliffs lit by overcast sky, visually grounding without competing.
- **Depth over borders.** Panels separated by background shade — the darker sea-cave panel recedes without a drawn line.
- **Rust-ochre cursor.** The cliff-rust caret is the warmest point in a cold frame — unmissable, anchored like stone.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
