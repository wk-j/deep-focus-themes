# Suspended Marl — แนวคิดชุดสี

## ที่มา

ธีมนี้มาจากภาพที่แสดงหน้าผาชายฝั่งสไตล์ไอร์แลนด์ ในวันที่หมอกลงจัด มีชายสวมเสื้อคลุมยืนอยู่ริมหน้าผา และมองไปยังก้อนหินทรงสามเหลี่ยมสามก้อนที่ลอยอยู่กลางอากาศเหนือทะเลสีเทาอมฟ้า แสงในภาพเป็นแสงสลัวที่กระจายทั่วจากหมอก ไม่มีแสงแดดโดยตรง

---

## แนวคิดหลัก

suspended-marl เป็นชุดสีสำหรับเขียนโค้ดที่ดึงแรงบันดาลใจมาจากบรรยากาศในภาพนั้น ความตึงเครียดหลักของธีมคือความขัดแย้งระหว่าง "ก้อนหินที่ควรตกลงมา แต่กลับลอยอยู่" ซึ่งแปลออกมาเป็นการใช้สีพื้นหลังเย็นๆ สีเทาอมน้ำเงิน คู่กับสีเน้นที่อุ่นกว่า

ธีมนี้แก้ความขัดแย้งนั้นด้วยสามแนวทาง:

- **พื้นหลังเย็น สมอสีอุ่น** — พื้นหลัง (`#181B1F`) มีโทนเทาอมน้ำเงิน เหมือนหินที่เปียกชื้นตลอดเวลา เคอร์เซอร์ใช้สีสนิมหิน (`#C47A5A`) ซึ่งเป็นสีอุ่นที่สุดในภาพ ทำให้มองเห็นได้ชัดเสมอ
- **สีฟ้าหมอกสำหรับฟังก์ชัน** — ฟังก์ชันใช้สี Fog Cyan (`#7DC4D4`) ซึ่งเป็นสีของแสงที่กระจายผ่านหมอกทะเล สว่างพอที่จะมองเห็นบนพื้นหลังมืด แต่ไม่จัดจ้านเกินไป
- **ความลึกจากระดับสี** — แถบด้านข้าง (sidebar) ใช้สีมืดกว่าพื้นที่แก้ไข ทำให้เกิดความลึกโดยไม่ต้องใช้เส้นขอบ เหมือนที่หน้าผาไกลออกไปดูเข้มกว่าในวันที่มีหมอก

---

## หลักการออกแบบ

### 1. สีที่ดึงมาจากบรรยากาศชายฝั่ง

สีเน้นทั้งหมดมาจากสิ่งที่เห็นในภาพจริงๆ:

- **Fog Cyan (`#7DC4D4`)** — สีของหมอกทะเลที่แพร่กระจายอยู่ในภาพ ใช้สำหรับฟังก์ชัน
- **Stone Mauve (`#9A8CB0`)** — สีของด้านเงาของก้อนหินที่ลอยอยู่ ใช้สำหรับ type
- **Cliff Grass (`#8AAA60`)** — สีหญ้าแห้งที่ขึ้นบนหน้าผาหินปูน ใช้สำหรับ string
- **Amber Dry (`#C4A050`)** — สีหญ้าที่แห้งสนิทและสีเหลืองอมส้มของหน้าผา ใช้สำหรับ constant และ attribute
- **Cliff Rust (`#C47A5A`)** — สีสนิมเหล็กบนหน้าผาหิน เป็นสีอุ่นที่สุดในชุด ใช้สำหรับ literal และเคอร์เซอร์

ไม่มีสีใดที่ถูกเลือกเพราะดูเหมาะกับธีม "มืด" โดยทั่วไป ทุกสีมาจากสิ่งที่มองเห็นในภาพต้นฉบับ

### 2. ลำดับความสำคัญของ syntax token

ลำดับนี้อิงจากว่า token ใดสำคัญที่สุดในการอ่านโค้ด:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| ฟังก์ชัน | Fog Cyan | `#7DC4D4` | สีที่แพร่หลายที่สุดในภาพ เหมาะกับ token ที่พบบ่อยที่สุด |
| Keyword | Coastal White | `#C8D8DC` | สีขาวอมเทาของหน้าผาในหมอก โครงสร้างหลักที่ไม่แย่งความสนใจ |
| Type | Stone Mauve | `#9A8CB0` | สีเงาของก้อนหิน ชัดเจนแต่ไม่โดดเด่นเกินไป |
| String | Cliff Grass | `#8AAA60` | สีอินทรีย์ที่ระดับพื้นดิน เนื้อหาที่จับต้องได้ |
| Attribute | Amber Dry | `#C4A050` | สีอุ่นอมเหลือง ใช้ขยายความหมาย |
| Literal / Number | Cliff Rust | `#C47A5A` | สีอุ่นที่สุด ค่าที่เป็นรูปธรรมที่สุด |
| Comment | Figure Dim | `#4A5060` | สีของเงาชายในภาพ มองเห็นได้แต่ไม่โดดเด่น |
| Error | Marl Red | `#C45860` | สีแดงเดียวในชุด แสดงข้อผิดพลาดได้ชัดโดยไม่ดูน่าตกใจ |

### 3. พื้นหลังที่มีความหมาย ไม่ใช่แค่สีดำ

พื้นหลังทุกระดับมีโทนน้ำเงินเทาเล็กน้อย ไม่ใช่สีดำสนิท เพราะหินชายฝั่งในภาพไม่เคยดูดำล้วนๆ

- **พื้นที่แก้ไข** (`#181B1F`) — สีมืดของหน้าผาในเงาหมอก
- **แถบด้านข้าง** (`#111417`) — มืดกว่าพื้นที่แก้ไข เหมือนส่วนลึกของถ้ำริมทะเล
- **Highlight / Selection** (`#252B30`) — สีที่สว่างขึ้นเล็กน้อยเมื่อเลือกข้อความ เหมือนหมอกจางลงชั่วขณะ
- **Error** ใช้ Marl Red (`#C45860`) — สีแดงอมส้มของรอยสนิมในหินปูน สื่อถึงปัญหาโดยไม่ทำให้ตื่นตระหนก

### 4. เคอร์เซอร์เป็นจุดยึด

เคอร์เซอร์ใช้ Cliff Rust (`#C47A5A`) เพราะเป็นสีอุ่นที่สุดในภาพต้นฉบับ โดดเด่นท่ามกลางพื้นหลังเย็นๆ ทำให้ตาหาตำแหน่งเคอร์เซอร์ได้ทันทีโดยไม่ต้องพยายาม

## ฟอนต์ที่เหมาะสม

ชุดสี suspended-marl ใช้ได้ดีกับ:

- **หลัก:** JetBrains Mono — เส้นสะอาด ตรงไปตรงมา เหมาะกับบรรยากาศที่เงียบและมีระเบียบของธีมนี้
- **ทางเลือก:** Fira Code — น้ำหนักเส้นหนากว่าเล็กน้อย เข้ากับโทนสีอุ่นปลายชุด

Ligature เช่น `=>`, `!==`, `>=` ช่วยให้โค้ดอ่านง่ายขึ้น เหมือนรอยต่อระหว่างชั้นหินที่เชื่อมโครงสร้างทั้งหมดเข้าด้วยกัน

## ความรู้สึกเมื่อใช้งานจริง

เปิดไฟล์ TypeScript ขึ้นมาในวันที่ท้องฟ้าเป็นสีเทา พื้นหลังของ editor จะให้ความรู้สึกเหมือนหินชื้นในแสงสลัว ไม่ดำสนิทแต่มืดพอที่จะไม่รบกวนสายตา ชื่อฟังก์ชันปรากฏเป็นสีฟ้าหมอก กระจายอยู่ทั่วโค้ดเหมือนหมอกที่คลุมอยู่ทุกที่ keyword สีขาวอมเทาอ่านได้โดยไม่ต้องสังเกต type สีเทาอมม่วงนั่งอยู่ในตำแหน่งของมันโดยไม่แย่งความสนใจ string สีหญ้าแห้งให้ความรู้สึกอินทรีย์และใกล้ชิดกว่า token อื่นๆ ตัวเลขและ literal สีสนิมหินมาพร้อมกับน้ำหนักที่เป็นรูปธรรม comment เลือนหายอยู่ที่ขอบ มองเห็นได้เมื่อต้องการ และเคอร์เซอร์สีสนิมอยู่นิ่งๆ รอจนกว่าจะขยับ เป็นจุดอุ่นเดียวในสนามเย็นที่ตาหาเจอได้เสมอ

นั่นคือ suspended-marl.
