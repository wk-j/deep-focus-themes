# Legacy Radiance

A dark coding color scheme using a **true-black void base** with a **three-frequency phosphor spectrum** — `#00C840` phosphor green functions, `#2A7EE8` cobalt blue keywords, `#00D4C8` ionized cyan types. Built for the long session that feels like archaeology.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#080808` | Void Black |
| Sidebar / Panels | `#040404` | Deep Void |
| Keywords | `#2A7EE8` | Cobalt |
| Functions | `#00C840` | Phosphor Green |
| Strings | `#00F0A0` | Plasma |
| Types & Tags | `#00D4C8` | Ionized Cyan |
| Numbers | `#80C898` | Pale Green |
| Constants & Booleans | `#D8F0E0` | Phosphor White |
| Variables | `#C8E0D0` | Screen Glow |
| Comments | `#304838` | Dead Phosphor |
| Attributes | `#7AE830` | Chartreuse |
| Errors | `#E83840` | Overload Red |

## Key Ideas

- **Three emission frequencies.** The palette is built on three distinct phosphor colors — the exact three emitted by a CRT monitor at different excitation levels: green (resting), blue (structural), cyan (peak charge).
- **Phosphor green as hero.** Functions use `#00C840` — the specific green of a P1 phosphor coating, the most common terminal screen material of the 1970s–80s. It glows; it does not merely color.
- **Cobalt keywords.** Structural keywords use `#2A7EE8` — the deep electric blue of the mid-ground code planes in the image, heavier and more authoritative than the bright green.
- **Depth over borders.** Panels separated by shade — `#080808` editor, `#040404` sidebar. At this level of darkness the difference is felt, not seen.
- **Ionized cyan cursor.** The `#00D4C8` caret is the brightest sustained point in the scheme — the exact color of a cursor block on a Wyse-60 terminal. Unmissable.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
