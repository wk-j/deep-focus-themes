# Stargazer

A dark coding color scheme using a **deep space midnight background** with a **galactic spectrum** — luminous cyan functions, golden keywords, green strings, sky-blue types, and amber literals. Built for long sessions under the stars.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#090E1A` | Deep Space |
| Sidebar / Panels | `#060A12` | Void Black |
| Keywords | `#F5C842` | Galactic Gold |
| Functions | `#29D7E8` | Nebula Cyan |
| Strings | `#4DC98A` | Aurora Green |
| Types & Tags | `#7EC8E3` | Star Field Blue |
| Numbers | `#E8922A` | Amber Core |
| Constants & Booleans | `#EDF4FF` | White Star |
| Variables | `#D6E8F5` | Moonlight |
| Comments | `#4D6D88` | Dark Matter |
| Attributes | `#7B8FD4` | Twilight Violet |
| Errors | `#E85555` | Red Giant |

## Key Ideas

- **Galactic color spectrum.** Each syntax role maps to a distinct hue drawn directly from the astrophotography — no two tokens share a color family.
- **Nebula cyan as hero.** Functions blaze in `#29D7E8`, the dominant luminous teal of the Milky Way arm — the most frequently called element deserves the brightest color.
- **Golden keywords mark structure.** `#F5C842` echoes the warm spiral core of the galaxy — bold, warm, and unmistakable amid the cool field.
- **Depth over borders.** Panels separated by shade, not lines — `#060A12` sidebar against `#090E1A` editor.
- **Cyan cursor.** The caret matches the hero function color — always visible, always anchored.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
