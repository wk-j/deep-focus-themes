# Stargazer — แนวคิดชุดสี

## วิสัยทัศน์

Stargazer เป็นชุดสีสำหรับเขียนโค้ดประสิทธิภาพสูงที่สร้างขึ้นสำหรับนักพัฒนาที่ทำงานลึกเข้าไปในดึก ปรัชญาคือ: ความเงียบของอวกาศลึกพบกับความเปล่งแสงของกาแล็กซีก้นหอย

ทำได้ผ่านสามแนวคิดที่เสริมกัน:

- **พื้นดำสนิท** — พื้นหลังดึงมาจากบริเวณมืดที่สุดของท้องฟ้ายามค่ำคืน (`#090E1A` ตัวแก้ไข, `#060A12` แผง) ไม่มีมลพิษแสงสะท้อนมาแข่งขันกับโค้ด
- **สีเน้นสเปกตรัมกาแล็กซี** — สีไวยากรณ์ที่ดึงมาโดยตรงจากภาพถ่าย: nebula cyan (`#29D7E8`) จากแขนทางช้างเผือก, galactic gold (`#F5C842`) จากแกนกลางก้นหอย, aurora green (`#4DC98A`) จากกระจุกดาว, sky blue (`#7EC8E3`) จากชั้นบรรยากาศด้านบน
- **คีย์เวิร์ดทอง ไวยากรณ์เย็น** — ทองอุ่นสำหรับคีย์เวิร์ดโครงสร้างสร้างความตัดกันของอุณหภูมิกับสเปกตรัมน้ำเงิน-เทียลของโค้ด ทำให้การควบคุมการไหลไม่อาจมองพลาด

ผลลัพธ์คือพื้นที่ทำงานที่รู้สึกเหมือนนั่งบนหลังคารถตอนเที่ยงคืน มองจักรวาลหมุนอยู่เบื้องบน

## หลักการออกแบบ

### 1. ภาพถ่ายดาราศาสตร์ในฐานะแหล่งชุดสี

สีเน้นทุกสีใน Stargazer สามารถติดตามกลับไปยังภาพอ้างอิงได้ สีเทียล-ไซยานที่โดดเด่นของแขนทางช้างเผือกกลายเป็นฟังก์ชัน — องค์ประกอบโค้ดที่คุณอ่านมากที่สุด ทองสว่าง-ขาวของแกนกลางก้นหอยกลายเป็นคีย์เวิร์ด — โครงสร้าง อุ่น มีแรงโน้มถ่วงสำคัญ แสงอำพันของขอบฟ้ากลายเป็นตัวเลขลิเทอรัล สีน้ำเงินเที่ยงคืนด้านบนกลายเป็นแท็กและโครงสร้าง HTML/XML ไม่มีสีที่ถูกประดิษฐ์ขึ้น ทั้งหมดถูกสังเกต

### 2. ลำดับชั้นทางสายตาที่ชัดเจนผ่านอุณหภูมิ

ชุดสีถูกแบ่งตามอุณหภูมิสีเพื่อเข้ารหัสความสำคัญทางความหมาย สิ่งนี้สร้างลำดับชั้นทางสายตาที่ชัดเจน:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| คีย์เวิร์ด | Galactic Gold | `#F5C842` | อุ่น; จุดยึดโครงสร้างที่นิยามการไหล |
| ฟังก์ชัน | Nebula Cyan | `#29D7E8` | สีหลัก; องค์ประกอบที่ใช้งานบ่อยที่สุด |
| สตริง | Aurora Green | `#4DC98A` | สะพานเย็น-อุ่น; เนื้อหาที่ต่างจากโค้ด |
| ชนิดข้อมูล | Star Field Blue | `#7EC8E3` | เย็น; มั่นคง เชิงนิยาม |
| ลิเทอรัล / ตัวเลข | Amber Core | `#E8922A` | สีเน้นอุ่น; ค่าที่เป็นรูปธรรมและทันที |
| ค่าคงที่ | White Star | `#EDF4FF` | ใกล้ขาว; คงที่ สว่าง สัมบูรณ์ |
| แอตทริบิวต์ | Twilight Violet | `#7B8FD4` | กลาง; ข้อมูลเมตา ไม่ใช่ตรรกะ |
| ตัวแปร | Moonlight | `#D6E8F5` | ใกล้ขาว; อ่านได้แต่เพลาลง |
| คอมเมนต์ | Dark Matter | `#4D6D88` | คอนทราสต์ต่ำ; อยู่แต่มองไม่เห็นเมื่อรีบ |
| ข้อผิดพลาด | Red Giant | `#E85555` | แดง; ไม่อาจเข้าใจผิด แต่ไม่น่าตกใจ |

### 3. Chrome น้อยที่สุด ความลึกสูงสุด

- **ไม่มีเส้นขอบแข็งกระด้าง** การแบ่งแผงมาจากการเปลี่ยนพื้นหลัง — `#090E1A` สำหรับตัวแก้ไข, `#060A12` สำหรับแถบด้านข้าง/แผง แค่พอแยกความลึกโดยไม่ลากเส้นที่มองเห็น
- **ตัวบ่งชี้แท็บที่ใช้งาน** เครื่องหมาย nebula cyan เพียงอย่างเดียวระบุไฟล์ปัจจุบัน
- **สถานะข้อผิดพลาดที่ยับยั้ง** ข้อผิดพลาดใช้ Red Giant (`#E85555`) — ผิดพลาดชัดเจน แต่อุ่นพอที่จะรู้สึกแก้ไขได้แทนที่จะเป็นหายนะ

### 4. เคอร์เซอร์ในฐานะจุดยึด

เคเรตและการเลือกต่างใช้ nebula cyan (`#29D7E8`) ต่อพื้นหลังดำสนิท เคอร์เซอร์ต้องมองเห็นได้ชัด — มันคือจุดแสงเดียวที่คุณกลับมาเสมอ การให้ตรงกับสีฟังก์ชันสร้างการเชื่อมโยงจิตใต้สำนึก: เคอร์เซอร์คือที่ที่ฟังก์ชันถัดไปเริ่มต้น

## การจับคู่แบบอักษร

ความเป็นเอกลักษณ์ของ "Stargazer" เสริมด้วยการจับคู่ชุดสีกับแบบอักษร monospace สมัยใหม่ที่รองรับลิเกเจอร์การเขียนโปรแกรม:

- **คำแนะนำหลัก:** JetBrains Mono — รูปทรงตัวอักษรสะอาด เพิ่มประสิทธิภาพสำหรับการอ่านโค้ดในขนาดเล็ก รูปแบบตัวอักษรยังคงชัดเจนต่อพื้นหลังมืดสุดโดยไม่เบลอ
- **ทางเลือก:** Fira Code — ชุดลิเกเจอร์ที่ยอดเยี่ยม (ลูกศร, การเปรียบเทียบ, lambda) น้ำหนักเส้นที่อุ่นกว่าเล็กน้อยเข้ากันได้กับกลยุทธ์คีย์เวิร์ดสีทอง

ลิเกเจอร์อย่าง `=>`, `!==`, และ `>=` ที่แสดงเป็นสัญลักษณ์เดี่ยวส่งเสริมความงามที่แม่นยำแบบหอสังเกตการณ์ที่ธีมมุ่งหมาย

## ความรู้สึกในทางปฏิบัติ

จินตนาการว่าคุณกำลังเปิดไฟล์ TypeScript ตอนตีหนึ่ง ห้องมืด:

- พื้นหลังคือ `#090E1A` — ไม่ใช่สีดำสนิท แต่เป็นสีกรมท่าลึกที่มีแรงดันซึ่งรู้สึกเหมือนท้องฟ้าหลังพระอาทิตย์ตกหนึ่งชั่วโมง
- ตาของคุณพบ `#F5C842` ทันที — `function`, `const`, `import`, เสาทองโครงสร้างของไฟล์
- ฟังก์ชันบานสะพรั่งด้วย `#29D7E8` องค์ประกอบที่สว่างที่สุดในตัวแก้ไข ติดตามทุกเส้นทางที่เรียกได้เหมือนแขนกาแล็กซีที่สว่าง
- สตริงนั่งอยู่ใน `#4DC98A` — สีเขียวเย็น มีรากฐาน เนื้อหาภายในโค้ดมากกว่าตัวโค้ดเอง
- ชนิดข้อมูลอ่านด้วย `#7EC8E3` สีฟ้าท้องฟ้าที่เงียบกว่า นิยามรูปร่างโดยไม่แข่งขันกับตรรกะที่ใช้งาน
- ตัวเลขและลิเทอรัลเรืองแสงด้วยอำพัน `#E8922A` อุ่นและเป็นรูปธรรม เหมือนไฟหน้ารถในทุ่งมืด
- ชื่อตัวแปรเป็น `#D6E8F5` — ชัดเจน ใกล้ขาว ไม่เรียกร้องความสนใจแต่อ่านได้เสมอ
- คอมเมนต์เป็น `#4D6D88` — พวกมันอยู่ในสสารมืดระหว่างโทเค็น มีเฉพาะเมื่อคุณมอง
- เคอร์เซอร์ของคุณลุกโชนด้วย `#29D7E8` ในโค้ด 10,000 บรรทัด คุณรู้ว่าคุณอยู่ที่ไหนเสมอ

นั่นคือ Stargazer.
