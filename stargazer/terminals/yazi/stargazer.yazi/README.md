# Stargazer

A dark Yazi flavor drawn from a midnight galaxy sky — deep space blues and teals, a warm galactic core, and silhouette-black grounds.

## Preview

![Stargazer preview](preview.png)

## Installation

```sh
ya pkg add wk/stargazer
```

## Usage

Set it as your dark flavor in `~/.config/yazi/theme.toml`:

```toml
[flavor]
dark = "stargazer"
```
