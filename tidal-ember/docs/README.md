# Tidal Ember

A dark coding color scheme using a **warm coastal base** with a **sunset-sand spectrum** — amber functions, golden strings, periwinkle types, burnt-ochre numbers, driftwood constants. Built for the last heat of a beach at golden hour.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#1A1410` | Wet Sand Dark |
| Sidebar / Panels | `#110E0A` | Tideline Deep |
| Keywords | `#E8D8C0` | Cream |
| Functions | `#D89840` | Tidal Amber |
| Strings | `#C4A050` | Sunset Gold |
| Types & Tags | `#8AA0C0` | Periwinkle |
| Numbers | `#C87830` | Burnt Ochre |
| Constants & Booleans | `#B8A888` | Driftwood |

| Comments | `#6A7888` | Steel Grey |
| Attributes | `#B8A888` | Driftwood |
| Errors | `#C04838` | Ember Red |

## Key Ideas

- **Sunset-sand spectrum.** Colors drawn from a beach at golden hour — amber glow on wet sand, gold light on the horizon, periwinkle sky overhead.
- **Amber hero.** Functions blaze in tidal amber (`#D89840`) — the dominant warm color of sun hitting retreating water.
- **Cream keywords.** Bold cream anchors mark structural flow without competing with the warm spectrum.
- **Depth over borders.** Panels separated by shade — tideline deep against wet sand dark.
- **Amber cursor.** The tidal amber caret (`#D89840`) is the last line of sunlight on the waterline — unmissable against the dark sand base.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
