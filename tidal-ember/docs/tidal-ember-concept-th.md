# Tidal Ember — แนวคิดชุดสี

## ที่มา

ภาพต้นแบบเป็นแมวขนสีเข้มนั่งอยู่บนหาดทรายกว้างในช่วงโกลเดนอาวร์ ดวงอาทิตย์อยู่ต่ำเหนือทะเล แสงกระจายผ่านหมอกและเมฆบางเป็นสีอำพัน-ทอง (`#D89840`) ที่แผ่ลงบนทรายเปียก แมวเกือบเป็นเงาดำ — ขนสีช็อกโกแลตเข้มตัดกับทรายที่เรืองแสง สายจูงสีฟ้าอ่อน (`#8AA0C0`) ทอดจากสายรัดของแมวมาทางกล้อง เป็นวัตถุสีเย็นเพียงชิ้นเดียวในภาพทั้งหมด ทรายเป็นเกรเดียนต์: ทรายแห้งสีเบจอุ่นใกล้ผู้ชม ทรายเปียกสีเทาน้ำตาลสะท้อนแสงใกล้แนวน้ำ มีแอ่งน้ำตื้นจับแสงแดดเหมือนเหรียญสีอำพันกระจายอยู่

---

## วิสัยทัศน์

Tidal Ember เป็นชุดสีสำหรับเขียนโค้ดที่ได้แรงบันดาลใจจากความอบอุ่นเฉพาะของหาดทรายไทยเวลาประมาณ 5 โมงครึ่ง — ชั่วโมงที่ดวงอาทิตย์อยู่ต่ำพอจนทุกอย่างกลายเป็นเกรเดียนต์ของสีอำพัน และคอนทราสต์เดียวที่เหลือคือระหว่างแสงอุ่นกับเงาดำ แรงตึงหลักคือระหว่างความอบอุ่นสีทองที่แผ่ไปทั่วภาพ กับโน้ตเย็นเพียงตัวเดียวของสายจูงสีฟ้าที่ตัดผ่าน

ธีมนี้แก้ปัญหาแรงตึงดังกล่าวผ่านการออกแบบ 3 จุด:

- **พื้นหลังสีทรายเข้ม** — สีพื้น (`#1A1410`) ไม่ใช่สีเทากลางหรือน้ำเงินดำ แต่เป็นสีน้ำตาลเข้มเฉพาะของทรายเปียกในเงา มีความอุ่นแม้ในความมืด เหมือนหาดทรายตอนค่ำที่ยังเก็บความร้อนของวันไว้ในพื้นดิน
- **สีอุ่นจากแสง** — Amber (`#D89840`), Gold (`#C4A050`), Ochre (`#C87830`), Driftwood (`#B8A888`), Cream (`#E8D8C0`) มาจากเกรเดียนต์ของแสงอาทิตย์ยามเย็นบนทรายและน้ำ ใช้กับ functions, strings, numbers, constants และ keywords
- **สีเย็นตัวเดียวจากสายจูง** — Periwinkle (`#8AA0C0`) คือสีฟ้าของสายจูงแมว วัตถุเย็นชิ้นเดียวในโลกอุ่น ใช้กับ types และ tags — คำจำกัดความเชิงโครงสร้าง สีเย็นหนึ่งสีท่ามกลางสีอุ่นหลายสีทำให้แยกชั้นโครงสร้างได้ทันที

ผลลัพธ์คือพื้นที่ทำงานที่ให้ความรู้สึกเหมือนบ่ายแก่ๆ บนทราย — อุ่น หนักแน่น ไม่เร่งรีบ — ตาแยก "โครงสร้าง" (เย็น) ออกจาก "เนื้อหา" (อุ่น) ได้โดยไม่ต้องใช้ความพยายาม

## หลักการออกแบบ

### 1. เกรเดียนต์โกลเดนอาวร์ — เส้นด้ายเย็นหนึ่งเส้นในผ้าอุ่น

ชุดสีนี้ไม่สมดุลโดยตั้งใจ: สีอุ่น 7 สีกับสีเย็น 1 สี อัตราส่วนนี้สะท้อนภาพถ่ายที่ทุกอย่าง — ฟ้า ทราย น้ำ แมว แม้แต่เงา — เป็นสีอุ่น มีแค่สายจูงที่เป็นสีฟ้า Functions (`#D89840`) เป็นสี Amber เพราะเป็นการกระทำที่สว่างที่สุดในโค้ด เหมือนจุดที่สว่างที่สุดในภาพคือตรงที่แสงแดดกระทบทรายเปียกโดยตรง Strings (`#C4A050`) เป็นสี Gold — นุ่มกว่าเล็กน้อย เหมือนแสงกระจายบนทรายแห้งที่ไกลจากแนวน้ำ Numbers (`#C87830`) เป็นสี Ochre — ความอุ่นเข้มข้นของดวงอาทิตย์ที่หักเหผ่านหมอก เข้มกว่าและส้มกว่าแสงทองทั่วไป Types (`#8AA0C0`) เป็นสี Periwinkle — สีเย็นตัวเดียวที่ทำหน้าที่โครงสร้างได้ทั้งหมด เพราะการโดดเดี่ยวทำให้มองเห็นชัดขึ้น ไม่ใช่ลดลง

### 2. ตรรกะเงาดำ — รูปทรงเข้ม พื้นสว่าง

แมวในภาพเกือบเป็นเงาดำ — ขนสีเข้มดูดซับแสงรอบตัว กลายเป็นรูปทรงที่ถูกกำหนดด้วยการไม่มีแสง Keywords เป็นสี Cream (`#E8D8C0`) ตัวหนา — เป็นรูปทรงโครงสร้างที่สว่าง กระดูกที่มองเห็นบนพื้นอุ่น เหมือนเส้นขอบของแมวที่อ่านออกได้เพราะทรายด้านหลังเรืองแสง ลำดับความสำคัญทางสายตา:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Keywords | Cream | `#E8D8C0` | ขอบสว่างของเงาดำ — โครงสร้าง ตัวหนา อุ่นเป็นกลาง |
| Functions | Tidal Amber | `#D89840` | ตัวเอก — แสงแดดตรงบนทรายเปียก สว่างที่สุด |
| Strings | Sunset Gold | `#C4A050` | แสงกระจายบนทรายแห้ง — ข้อมูล ตรงไปตรงมา หนักแน่น |
| Types | Periwinkle | `#8AA0C0` | สายจูง — เส้นด้ายเย็นเส้นเดียวที่กำหนดโครงสร้าง |
| Numbers | Burnt Ochre | `#C87830` | แดดเข้มข้นผ่านหมอก — เฉพาะเจาะจง ลึก อุ่น |
| Constants | Driftwood | `#B8A888` | ไม้ลอยน้ำบนแนวน้ำ — คงที่ มีตัวตน นุ่ม |
| Comments | Steel Grey | `#6A7888` | มหาสมุทรที่ขอบฟ้า — มองเห็นได้แต่ถอยลง เย็นเป็นกลาง |
| Variables | Shore White | `#D8CCBA` | ทรายแห้งในแสงเปิด — พื้นผิวเริ่มต้น อ่านง่าย อุ่น |

### 3. พื้นหลังเป็นเวที ไม่ใช่ความว่างเปล่า

พื้นหลัง `#1A1410` เป็นสีเข้มอมน้ำตาลที่อ่านเป็น "วัสดุ" ไม่ใช่ "ความว่าง" เป็นสีของทรายเปียกหลังน้ำลง: เข้ม อุ่น มีความอุดมเล็กน้อยที่สีดำสนิทไม่มี โทนอุ่นเชื่อมพื้นหลังกับสี Amber ของ functions และสี Gold ของ strings ทำให้ทุกอย่างอยู่บนหาดเดียวกัน รายละเอียด:

- **พื้นผิว editor** (`#1A1410`) — ทรายเปียกในเงาเมฆ มืดพอสำหรับใช้งานนานๆ อุ่นพอให้รู้สึกเหมือนดิน ไม่ใช่จอ
- **Panel/sidebar** (`#110E0A`) — แนวน้ำตอนค่ำ ตรงที่ทรายเปียกพบทรายแห้งในเงาลึกที่สุด ถอยลง เป็นพื้นฐาน
- **Highlight** (`#2A2218`) — แอ่งน้ำตื้นจับแสงอำพัน การเลือกข้อความยกตัวด้วยความอุ่น ไม่ใช่ความสว่าง
- **Errors** ใช้สี Ember Red (`#C04838`) — สีถ่านในกองไฟชายหาด มองเห็นชัดและเร่งด่วนแต่อยู่ในโลกความร้อนเดียวกับ Amber และ Ochre

### 4. เคอร์เซอร์เป็นจุดยึด

เคอร์เซอร์ใช้สี Tidal Amber (`#D89840`) ในภาพ จุดที่สว่างที่สุดคือตรงที่แดดต่ำสะท้อนจากแผ่นน้ำบางๆ ที่น้ำลงทิ้งไว้ — แถบแสงทองเข้มข้นที่ตาหลีกเลี่ยงไม่ได้ เคอร์เซอร์สี Amber คือแถบนั้น: เส้นแสงอุ่นคมชัดบนพื้นทรายเข้ม ในชุดสีที่เกือบทุกอย่างเป็นสีอุ่น เคอร์เซอร์ชนะด้วยการเป็นสีอุ่นที่อิ่มตัวที่สุด — การแสดงออกบริสุทธิ์ที่สุดของโกลเดนอาวร์ ตาของคุณกลับมาหาจุดนี้เสมอ เหมือนตาที่กลับไปหาแสงแดดสะท้อนบนน้ำ

## การจับคู่แบบอักษร

ชุดสี "Tidal Ember" เข้าคู่กับ:

- **หลัก:** JetBrains Mono — ความหนาเส้นสม่ำเสมอทำให้สีอุ่นหลายสีอ่านได้ชัดโดยไม่เบลอเข้าหากันที่ขนาดเล็ก ความแม่นยำของตัวอักษรยึดความอุ่นของชุดสีไว้กับพื้นดิน เหมือนสายรัดที่ยึดแมวไว้กับหาดทราย
- **ทางเลือก:** Fira Code — ปลายตัวอักษรนุ่มกว่าเล็กน้อย ทำให้สี Amber และ Gold หายใจได้มากขึ้น แลกความแม่นยำกับคุณภาพที่หลวมกว่าเล็กน้อยของแสงบ่ายแก่

Ligatures อย่าง `=>`, `!==` และ `>=` เสริมจังหวะที่ไม่เร่งรีบของธีม — อ่านเป็นท่าทางเดียวแทนที่จะเป็นลำดับ เหมือนคลื่นที่ถอยกลับผ่านทรายเป็นการเคลื่อนไหวต่อเนื่องหนึ่งเดียว ไม่ใช่ขั้นตอนหลายขั้น

## ความรู้สึกในการใช้งานจริง

ลองจินตนาการเปิดไฟล์ Rust ตอนค่ำ มีแสงอุ่นจากหน้าต่างด้านหลัง พื้น editor ไม่ใช่สีดำ — เป็นสีน้ำตาลเข้มอุ่นของทรายหลังน้ำลง พื้นผิวที่เก็บความร้อน ตาเคลื่อนไปที่ functions สี Amber ก่อน: สีอุ่นที่สว่างที่สุด แถบแสงแดดบนน้ำลง ดึงคุณเข้าไปในโค้ดเหมือนแสงสะท้อนบนหาดเลนดึงตาไปทางขอบฟ้า ระหว่าง functions strings สี Gold นั่งมั่นคง — แสงกระจายของทรายแห้ง พาข้อมูลจริงของโปรแกรมในโทนอุ่นหนักแน่น Types ปรากฏในสี Periwinkle โน้ตเย็นเพียงตัวเดียวในทุ่งอุ่น และการโดดเดี่ยวนี้ทำให้อ่านได้ทันที — คุณรู้เสมอว่าคำจำกัดความโครงสร้างอยู่ตรงไหนเพราะมันเป็นสิ่งเดียวที่เป็นสีฟ้าบนผ้าใบอุ่น Numbers เผาไหม้ลึกกว่าในสี Ochre เข้มข้นและเฉพาะเจาะจง สีของดวงอาทิตย์ที่มองผ่านหมอก Constants พักอยู่ในสี Driftwood — นุ่ม มีตัวตน สีซีดของไม้ที่อยู่บนหาดนานจนฟอกขาว Keywords สี Cream ยึดโครงสร้างไว้ด้วยความอุ่นเงียบๆ หนาแต่ไม่มีสี ขอบสว่างของทุกเงาดำ Comments ถอยลงไปในสี Steel Grey มหาสมุทรที่เส้นขอบฟ้า อ่านได้เมื่อมองแต่ไม่แข่ง และเคอร์เซอร์กะพริบในสี Tidal Amber — สีเดียวกับแถบแสงแดดบนน้ำที่ตาของคุณหาเจอก่อนเสมอ
