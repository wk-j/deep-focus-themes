# Desert Monolith

A dark coding color scheme using a **deep desert night** with a **warm-to-cool spectrum** — sky-blue functions, sandy strings, hazy blue-grey types, ochre tags, sage literals. Built for long sessions that feel ancient and focused.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#1A1510` | Desert Night |
| Sidebar / Panels | `#130F0B` | Deep Shadow |
| Keywords | `#B8C4C8` | Pale Rock |
| Functions | `#7BB5D0` | Sky Blue |
| Strings | `#C4A86A` | Desert Sand |
| Types & Tags | `#8A9EAD` | Mountain Haze |
| Numbers | `#7A9E84` | Desert Sage |
| Constants & Booleans | `#E8E0D0` | Worn Ivory |
| Variables | `#D4C9B8` | Sandstone |
| Comments | `#6E5E4A` | Buried Stone |
| Attributes | `#D4902A` | Amber Light |
| Tags | `#B87840` | Ochre Rock |
| Errors | `#CC4A3A` | Rust Scar |

## Key Ideas

- **Warm earth meets cold sky.** The palette spans the exact temperature gradient in the image — ochre sands at the foreground, pale silvery stone faces, cool blue sky at distance.
- **Sky blue as hero.** Functions use `#7BB5D0` — the open sky above the monolith, the most visually expansive element.
- **Pale rock keywords.** Structural keywords use `#B8C4C8`, the weathered light face of the standing stone — neutral, authoritative, ever-present.
- **Depth over borders.** Panels separated by shade, not lines — `#1A1510` editor vs. `#130F0B` sidebar.
- **Amber cursor.** The caret uses `#D4902A`, the warm afternoon light catching stone edges. Unmissable in the dark.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
