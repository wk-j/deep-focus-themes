# Lantern Static — แนวคิดการออกแบบธีม

## ที่มา

ภาพอ้างอิงคือตรอกในโตเกียวยามค่ำคืน — ซ้ายมือเป็นร้านราเมนที่มีโคมกระดาษสีแอมเบอร์แขวนอยู่เหนือประตู ผนังคอนกรีตเก่าๆ มีลายสเปรย์อักษรคันจิ และตู้ขายตั๋วอาหารริมทาง ขวามือเป็นร้านซ่อมอิเล็กทรอนิกส์ที่เต็มไปด้วยลังไม้ สายเคเบิล และอุปกรณ์เก่า พร้อมหุ่นยนต์เชื่อมโลหะที่พ่นแสงพลาสมาสีฟ้า-ขาว ป้ายนีออนสีแดงอมม่วงเรืองแสงอยู่เหนือทางเข้า พื้นตรอกและความมืดระหว่างสองร้านดูดซับแสงทั้งสองไว้โดยไม่แยกแยะ

---

## วิสัยทัศน์

Lantern Static เป็นธีมสีสำหรับเขียนโค้ดที่ดึงแรงบันดาลใจจากความตึงเครียดระหว่างความอบอุ่นของโคมไฟโบราณกับความเย็นจากเครื่องจักรในพื้นที่เดียวกัน ความขัดแย้งหลักคือ โคมกระดาษ กับ หัวเชื่อมพลาสมา — แสงรูปแบบเก่าแก่ที่สุดและใหม่ที่สุดอยู่ในตรอกเดียวกัน

ธีมนี้แก้ความขัดแย้งนั้นด้วยการตัดสินใจสามอย่าง:

- **ฮีโร่อบอุ่น โครงสร้างเย็น** — ฟังก์ชันใช้สีแอมเบอร์ (`#E8922A`) ซึ่งเป็นสีของโคมกระดาษ ส่วน type และ tag ใช้สีพลาสมาฟ้า (`#52C8F0`) จากหุ่นยนต์ ทั้งสองสีชัดเจนแต่อยู่คนละฝั่งของอุณหภูมิสี จึงไม่แย่งความสนใจกัน
- **String สีแดงเหมือนป้ายนีออน** — string literal ใช้สีแดงเข้ม (`#E8314A`) อ้างอิงจากป้ายคันจิเหนือทางเข้าตรอก เป็นสีที่ประกาศตรงๆ เหมือนป้ายร้าน
- **โทนกลางสีคอนกรีต** — ผนังตรอกสีม่วง-เทาหม่นกลายเป็นสีของ comment (`#8B72BE`) และ foreground ที่หรี่ลง ไม่อบอุ่น ไม่เย็น — เป็นสีตกค้างของแสงทั้งสองชนิด

## หลักการออกแบบ

### 1. แกนโคมไฟ-พลาสมา

สีหลักทุกตัวในธีมนี้อยู่บนสเปกตรัมระหว่างแสงอุ่นกับแสงเย็น สีแอมเบอร์ (`#E8922A`) และเหลืองโคมไฟ (`#F5C842`) อยู่ฝั่งอุ่น ครอบคลุม function, constant, attribute, number สีพลาสมาฟ้า (`#52C8F0`) และหมอกเทา (`#8FAABB`) อยู่ฝั่งเย็น ครอบคลุม type, tag, constructor สีแดง (`#E8314A`) เป็นสีของป้ายภาษา ไม่ได้อยู่ฝั่งไหน

### 2. ลำดับความสำคัญของ syntax

ลำดับสีสะท้อนความสำคัญในฉากจริง:

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Keywords | Smoke White | `#F2EEF8` | โครงสร้างหลัก เหมือนเสาคอนกรีต — เป็นกลาง มีอยู่ทุกที่ |
| Functions | Lantern Amber | `#E8922A` | จุดอบอุ่นที่สุด เหมือนโคมกระดาษที่สว่างที่สุดในฉาก |
| Strings | Neon Crimson | `#E8314A` | ป้ายนีออนคันจิ — ภาษาที่ประกาศตรงๆ |
| Types | Plasma Blue | `#52C8F0` | แสงพลาสมาหุ่นยนต์ — แม่นยำ เย็น อ่านได้โดยเครื่อง |
| Constants | Sign Yellow | `#F5C842` | ป้ายราคาในหน้าต่างร้าน |
| Numbers | Rust Glow | `#C46B3A` | โลหะเกรอะภายใต้แสงโคม |
| Comments | Indigo Haze | `#8B72BE` | แสงนีออนดูดซึมในปูนผนัง — มีอยู่แต่ไม่ถูกอ่าน |
| Variables | Dusty White | `#D8D0E8` | ระดับแสงรอบข้าง — ทุกอย่างที่ไม่ได้ถูกส่องแสงเฉพาะ |

### 3. พื้นหลังเป็นเวที ไม่ใช่ความว่าง

- **พื้นหลังหลัก** (`#0E0D15`) — ความมืดของคอนกรีตระหว่างสองร้านที่มีแสง ไม่ใช่สีดำล้วน แต่มีน้ำเงินจางๆ บอกว่ามีวัสดุอยู่
- **แผงข้าง/sidebar** (`#090810`) — ภายในตรอกที่ลึกกว่า แสงโคมและแสงพลาสมาไม่ถึง
- **Highlight** (`#1A1826`) — แสงสะท้อนบนผนังเมื่อแสงสองแหล่งกระทบพื้นที่เดียวกัน ใช้สำหรับ cursorline และ selection
- **Error** ใช้สีแดงเตือนภัย (`#E83040`) — เหมือนสติกเกอร์เตือนไฟบนกล่องฟิวส์

### 4. Cursor เป็นหมุดยึด

Cursor ใช้สีพลาสมาฟ้า (`#52C8F0`) ในฉาก หัวเชื่อมพลาสมาเป็นแหล่งแสงเดียวที่เคลื่อนที่ได้ — มันติดตามงานของหุ่นยนต์เสมอ ในโปรแกรมแก้ไข cursor ทำหน้าที่เดียวกัน: บอกว่างานเกิดขึ้นที่ไหน สีฟ้านี้เป็นสีเย็นที่สุดในพาเลตต์ จึงมองเห็นได้ชัดเจนแม้อยู่ท่ามกลางสีแอมเบอร์และขาวกลาง

## การจับคู่ฟอนต์

ธีม "Lantern Static" เข้ากันดีกับ:

- **หลัก:** JetBrains Mono — รูปตัวอักษรเชิงกล เหมาะกับบรรยากาศร้านซ่อมและสีแอมเบอร์ที่มีความอิ่มตัวเพียงพอจะใช้กับรูปทรงเส้นสะอาด
- **ทางเลือก:** Fira Code — น้ำหนักเส้นโค้งกว่าเล็กน้อย ทำให้สีแอมเบอร์ดูอบอุ่นขึ้น ใกล้โคมไฟมากขึ้น

Ligature อย่าง `=>`, `!==`, `>=` ช่วยเสริมบรรยากาศเชิงกลของธีม — เหมือนวงจรที่ต่อครบและทำงานได้

## ความรู้สึกขณะใช้งานจริง

จินตนาการว่าคุณเปิดไฟล์ TypeScript ที่ซับซ้อนตอนสี่ทุ่ม พื้นหลัง `#0E0D15` ตั้งเงียบ — มืดพอที่จะโฟกัสแต่ไม่โล่งเหมือนหน้าจอที่ไม่ได้ออกแบบมา keywords ปรากฏก่อน: `const`, `function`, `return` — ขาวหนาชัดเจน เป็นโครงกระดูกของไฟล์ จากนั้น function ต่างๆ สว่างขึ้นด้วยสีแอมเบอร์ ชื่อแต่ละฟังก์ชันเรืองแสงด้วยความอบอุ่นของสิ่งที่กำลังลุกไหม้อยู่ ไม่ใช่แสงไฟฟ้า String ตามมาด้วยสีแดงเข้ม สั้น ประกาศตรง เหมือนป้ายเหนือตรอก Type ปรากฏเป็นสีฟ้าเย็น แม่นยำและอ่านได้โดยเครื่อง Comment จืดลงเป็นสีม่วงอินดิโก มีอยู่แต่ไม่รบกวน และตลอดเวลา cursor สีพลาสมาฟ้าเคลื่อนไหวอยู่ — จุดเดียวในฉากที่ทั้งเย็นและมีชีวิต บอกว่างานกำลังเกิดขึ้นที่นี่

นั่นคือ Lantern Static.
