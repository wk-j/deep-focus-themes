# Lantern Static

A dark coding color scheme drawn from a **Tokyo backstreet at night** — amber functions burning like a paper lantern, plasma-blue types from machine discharge, crimson strings like neon kanji signs, violet-grey comments dissolving into concrete shadow. Built for sustained focus in cluttered, layered code.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0E0D15` | Alley Dark |
| Sidebar / Panels | `#090810` | Deep Corridor |
| Keywords | `#F2EEF8` | Smoke White |
| Functions | `#E8922A` | Lantern Amber |
| Strings | `#E8314A` | Neon Crimson |
| Types & Tags | `#52C8F0` | Plasma Blue |
| Numbers | `#C46B3A` | Rust Glow |
| Constants & Booleans | `#F5C842` | Lantern Yellow |
| Variables | `#D8D0E8` | Dusty White |
| Comments | `#8B72BE` | Indigo Haze |
| Attributes | `#F5C842` | Sign Yellow |
| Errors | `#E83040` | Alarm Red |

## Key Ideas

- **Warm-cold tension.** Amber and crimson from the lantern and neon signs oppose the machine-blue plasma of the robot — the same opposition runs through functions vs. types.
- **Amber as hero.** Functions in `#E8922A` — the exact color of a paper lantern in a dark alley, the warmest and most alive point in the frame.
- **White keyword anchors.** Structural keywords in near-white bold stand apart from all color — they are the load-bearing walls of the code.
- **Depth over borders.** Panels separated by shade, not lines.
- **Plasma-blue cursor.** `#52C8F0` — unmissable, the color of the welding robot's discharge, always the hottest point in the scene.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
