# Twilight Overclock

A dark coding color scheme using a **Deep Forest** base with **Bioluminescent** accents. Built for long sessions — low fatigue, high clarity.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0B1215` | Deep Charcoal-Teal |
| Sidebar / Panels | `#070B0D` | Void Teal |
| Keywords | `#7FB069` | Sage Moss |
| Functions | `#00F5FF` | Electric Cyan |
| Strings & Literals | `#FFB347` | Honey Amber |
| Variables | `#E0E0E0` | Mist White |
| Comments | `#4A5568` | Slate Shadow |
| Types | `#5B9BD5` | Glacier Blue |
| Errors | `#FF5F5F` | Soft Coral |

## Key Ideas

- **Depth over borders.** Panels separated by shade, not lines.
- **Cyan cursor.** Always visible, never lost.
- **Soft errors.** Coral instead of emergency red.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
