# Tallow Light — แนวคิดของธีมสี

## ที่มา

ภาพอ้างอิงเป็นฟาร์มบนที่ราบในช่วงเวลาระหว่างพระอาทิตย์ตกและความมืดสมบูรณ์ บ้านที่เป็นเงาดำตั้งอยู่ท่ามกลางขอบฟ้าสีส้มอำพัน เมฆพายุทับซ้อนกัน และดวงจันทร์ส่องแสงออกมาจากช่องเมฆ กังหันลมยืนเป็นโครงเหล็กด้านซ้าย หน้าต่างของบ้านเปล่งแสงสีอำพันทองอุ่น — เป็นจุดเดียวในภาพที่บอกว่ามีชีวิตอยู่ภายใน หญ้าในพื้นหน้าเป็นสีฟางแห้ง ภูเขาในระยะไกลเป็นสีน้ำเงินเทาหิมะ และท้องฟ้าไล่สีจากสีเทาดำของพายุไปสู่แถบสีส้มอมน้ำตาลที่ขอบฟ้า

---

## วิสัยทัศน์

Tallow Light เป็นธีมสีที่ได้แรงบันดาลใจจากช่วงเวลาที่แสงเทียนในฟาร์มกลายเป็นสิ่งที่สว่างที่สุดในทุ่งกว้าง แรงตึงเครียดหลักคือความขัดแย้งระหว่างสีอบอุ่น (สีเทียน สีอำพัน ขอบฟ้าส้ม) กับสีเย็น (สีหินภูเขา สีท้องฟ้าพายุ สีดินมืด)

ธีมแก้ไขความขัดแย้งนี้ด้วยสามการตัดสินใจ:

- **หน้าต่างเป็นจุดศูนย์กลาง** — ฟังก์ชันใช้ Tallow (`#E8B050`) สีอำพันอุ่นของหน้าต่างบ้านในภาพ เป็นสีที่สายตาหาก่อนและกลับมาหาเสมอ
- **Ember สำหรับ keywords** — keywords ใช้ `#D45A20` สีส้มน้ำตาลของขอบฟ้าใต้เมฆพายุ — เป็นไฟที่ให้ทิศทาง ไม่ใช่ความร้อน
- **Slate สำหรับ types** — types ใช้ `#7A8EA8` สีภูเขาหิมะในระยะไกล — เย็น รับน้ำหนักโครงสร้าง

---

## หลักการออกแบบ

### 1. ระดับอุณหภูมิ: Tallow → Ember → Straw

สีอบอุ่นสามสีเรียงตามอุณหภูมิลดลง:
- **Tallow `#E8B050`** → ฟังก์ชัน (แสงหน้าต่างที่ใกล้ที่สุดและอบอุ่นที่สุด)
- **Ember `#D45A20`** → keywords (แนวขอบฟ้า — ต่ำกว่าและเย็นกว่า แต่ยังคงเป็นทิศทาง)
- **Straw `#A07838`** → strings และตัวเลข (หญ้าฟางแห้ง — อินทรีย์ ไม่ส่องแสง)

### 2. ระยะไกลที่เย็น: Slate, Mauve, Moonlight

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Types | Slate | `#7A8EA8` | สีภูเขาหิมะ — โครงสร้าง ไกล เย็น |
| Attributes | Mauve | `#8A7888` | สีม่วงคล้ำของขอบฟ้าด้านขวา — ตกแต่ง ถดถอย |
| Constants | Moonlight | `#9AAEC0` | สีน้ำเงินขาวซีดของแสงจันทร์ — คงที่ เย็น |
| Comments | Deep Ochre | `#4A4438` | ดินมืดระหว่างหญ้า — มีอยู่แต่ต่ำกว่าระดับสายตา |
| Errors | Coal Red | `#C03828` | สีแดงจากฝั่งไฟ ไม่ใช่ท้องฟ้า |

### 3. พื้นหลังเป็นเวที ไม่ใช่ความว่างเปล่า

พื้นหลังมีโทนสีน้ำตาลอุ่นจากดินและหญ้า ไม่ใช่สีดำน้ำเงิน:

- **Editor surface** (`#0E0D0B`) — ความมืดของเงาบ้านบนขอบฟ้า: มืดแบบดิน ไม่ใช่ดำล้วน
- **Panel/sidebar** (`#090807`) — เงาลึกใต้เมฆที่ไม่มีแสงส่องถึง
- **Highlight** (`#1A1710`) — แสงที่หยดออกมาจากหน้าต่างบ้านตกลงบนหญ้าใกล้ๆ
- **Errors** ใช้ Coal Red (`#C03828`) — สีแดงที่มาจากฝั่งไฟ อ่านเป็นสัญญาณอันตรายโดยไม่สว่างจนน่าวิตก

### 4. Cursor เป็นจุดยึด

Cursor ใช้ Tallow (`#E8B050`) — เหมือนหน้าต่างที่สว่างในภาพ ไม่ว่าท้องฟ้าจะมืดแค่ไหน คุณหาบ้านได้เสมอจากแสงหน้าต่าง Cursor อำพันทำหน้าที่เดียวกัน — ไม่ว่าโค้ดจะหนาแน่นแค่ไหน จุด tallow เดียวบอกตำแหน่งของคุณ

## การจับคู่ฟอนต์

ธีม "Tallow Light" เข้ากันได้ดีกับ:

- **หลัก:** JetBrains Mono — รูปทรงเรขาคณิตหนาแน่นเหมาะกับชุดสีที่สร้างจากความอบอุ่นที่ควบคุมได้และความอดทนทางธรณีวิทยา ฟังก์ชันสีอำพันเรืองแสงบนฟอนต์นี้เหมือนแสงตะเกียงที่คงที่
- **ทางเลือก:** Fira Code — โครงสร้างที่ไหลลื่นกว่าเหมาะกับองค์ประกอบอินทรีย์ของชุดสี: strings สีฟาง, variables สีที่ราบ, attributes สีม่วง

Ligatures เช่น `=>`, `!==`, `>=` ลดความวุ่นวายของ operators และทำให้ฟังก์ชัน tallow และ keywords ember เป็นจุดสำคัญทางความร้อนบนแต่ละบรรทัด

## ความรู้สึกเมื่อใช้งานจริง

ลองนึกภาพการเปิดไฟล์ TypeScript ตอนสี่ทุ่มพร้อมเสียงฝนนอกหน้าต่าง: พื้นหลัง (`#0E0D0B`) วางตัวเหมือนห้องที่เปิดไฟมาพักหนึ่งจนความมืดข้างนอกสมบูรณ์แบบ สายตาของคุณพบการประกาศฟังก์ชันแรกทันที — tallow amber `#E8B050` อบอุ่นและมั่นคง สิ่งเดียวที่เรืองแสงในทิศนั้น อ่านลงมาในเนื้อหาฟังก์ชัน keywords ember `#D45A20` ทำหน้าที่เครื่องหมายจุดแยกและการ return — เหมือนเส้นขอบฟ้า ต่ำกว่าแสงหลักแต่บอกทิศทาง การประกาศ type ปรากฏในสี slate `#7A8EA8` เย็นและรับน้ำหนักโครงสร้าง Strings อ่านในสีฟาง `#A07838` แห้งและอินทรีย์ Constants ลอยอยู่ใน moonlight `#9AAEC0` ซีดและตายตัว comments แทบมองไม่เห็น ถอยอยู่ในดินระหว่างหญ้า และตลอดทั้งหมด จุด amber เดียว — cursor tallow ที่ `#E8B050` — ยึดตำแหน่งในความมืด เป็นสิ่งเดียวที่คุณต้องหาเพื่อรู้ว่าคุณอยู่ที่ไหน

นั่นคือ Tallow Light.
