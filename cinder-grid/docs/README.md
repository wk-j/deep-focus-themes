# Cinder Grid

A dark coding color scheme using a **mosaic-tile control room at night** as its reference — ember-orange functions, steel-blue types, amber keywords, moss-green strings, and scattered city-light accents against a near-black charcoal-navy surface. Built for long sessions in the glow.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0B0F1A` | Charcoal Navy |
| Sidebar / Panels | `#070B14` | Deep Station |
| Keywords | `#D4A827` | Amber |
| Functions | `#E8721A` | Ember |
| Strings | `#5FAD6E` | Moss |
| Types & Tags | `#4D8FCC` | Tile Blue |
| Numbers | `#C8901A` | Gold |
| Constants & Booleans | `#C8901A` | Gold |
| Variables | `#D8DDE8` | Near White |
| Comments | `#4E5A6E` | Muted Slate |
| Attributes | `#7BB5D0` | Sky Blue |
| Errors | `#D45C5C` | Coral Red |

## Key Ideas

- **Ember as the hero.** Function calls burn in `#E8721A` — the dominant color of the left monitor's cascading code, the visual anchor of the image.
- **Amber keywords.** `#D4A827` echoes the floating gold squares drifting across the mosaic ceiling — structural markers that feel like embedded light.
- **Tile blue for types.** `#4D8FCC` matches the electric-blue highlights in the tile wall, grounding type declarations in the room's material texture.
- **Depth over borders.** Panels separated by shade, not lines. `#070B14` recedes behind `#0B0F1A`.
- **Ember cursor.** `#E8721A` caret — the same color as functions, unmissable against the dark navy ground.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
