# Cinder Grid — แนวคิดของธีมสี

## ที่มา

ภาพอ้างอิงเป็นโต๊ะทำงานของโปรแกรมเมอร์ในเวลากลางคืน มองจากด้านบนและด้านหลัง จอทางซ้ายแสดงโค้ดสีส้มแดงหนาแน่น คล้ายโลหะหลอมเหลวที่เรืองแสง ห้องทำงานประกอบด้วยกระเบื้องโมเสกขนาดเล็กหลากเฉดสีน้ำเงินเหล็กและถ่าน พร้อมจุดแสงสีอำพันกระจายอยู่บนผนังราวกับพิกเซลที่หลุดออกมาจากหน้าจอ จอทางขวาแสดงภาพเมืองจากที่สูง มีจุดแสงสีขาวและฟ้าซายันนับพันจุดลอยอยู่ในความมืด

---

## วิสัยทัศน์

Cinder Grid เป็นธีมสีสำหรับเขียนโค้ดที่ได้แรงบันดาลใจจากห้องควบคุมกระเบื้องโมเสกยามดึก แรงตึงเครียดหลักของธีมคือความขัดแย้งระหว่างสีที่ให้ความรู้สึกร้อน (ember, amber, gold) กับสีที่ให้ความรู้สึกเย็นและเป็นโครงสร้าง (steel blue, navy)

ธีมแก้ไขความขัดแย้งนี้ด้วยสามการตัดสินใจออกแบบ:

- **ลำดับชั้นสี ember** — ฟังก์ชันใช้สี `#E8721A` ซึ่งเป็นสีที่โดดเด่นที่สุดในจอซ้ายของภาพ สายตาของผู้ดูภาพพุ่งไปที่จุดนี้ก่อน เช่นเดียวกับสายตาของนักพัฒนาที่มองหาฟังก์ชันก่อนเสมอ
- **สีกระเบื้องสำหรับโครงสร้าง** — ประเภทข้อมูล (types) และ tags ใช้สี `#4D8FCC` ที่สกัดมาจากไฮไลต์สีน้ำเงินไฟฟ้าบนกำแพงกระเบื้อง
- **ความมืดที่มีความลึก** — พื้นหลัง `#0B0F1A` ไม่ใช่สีดำล้วน แต่เป็นสีที่ได้จากความมืดระหว่างกระเบื้องที่มีแสงส่อง

## หลักการออกแบบ

### 1. สเปกตรัมอุณหภูมิ: Ember ถึง Amber ถึง Gold

สำเนียงสีอบอุ่นสามสี — ember (`#E8721A`), amber (`#D4A827`), gold (`#C8901A`) — เรียงตามระดับอุณหภูมิลดลง:
- **Ember** → ฟังก์ชัน (สิ่งที่ "เผาไหม้" มากที่สุดในการคำนวณ)
- **Amber** → keywords (เครื่องหมายโครงสร้าง เหมือนจุดแสงอำพันบนผนัง)
- **Gold** → ตัวเลขและค่าคงที่ (ค่าที่ตายตัว ไม่เปลี่ยนแปลง)

### 2. กริดเย็น: Type, Attribute, Sky

| บทบาท | สี | Hex | เหตุผล |
|---|---|---|---|
| Types | Tile Blue | `#4D8FCC` | สีเดียวกับไฮไลต์บนกระเบื้อง — โครงสร้างรับน้ำหนัก |
| Attributes | Sky Blue | `#7BB5D0` | สีอ่อนกว่า คล้ายท้องฟ้าที่สะท้อนในกระเบื้อง |
| Tags | Tile Blue | `#4D8FCC` | Tags ใน HTML/XML เป็นโครงสร้างเหมือน type |
| Strings | Moss | `#5FAD6E` | สีอินทรีย์เพียงสีเดียวในชุดสี — strings เก็บข้อความของมนุษย์ |
| Comments | Muted Slate | `#4E5A6E` | เงาระหว่างกระเบื้อง — มีอยู่แต่ไม่แย่งความสนใจ |

### 3. พื้นหลังเป็นเวที ไม่ใช่ความว่างเปล่า

พื้นหลังมีสี่ระดับ แต่ละระดับให้ความลึกต่างกัน:

- **พื้นที่ editor** (`#0B0F1A`) — ความมืดระหว่างกระเบื้องที่มีแสง มีโทนสีน้ำเงินบางๆ
- **Panel/sidebar** (`#070B14`) — มืดกว่าหนึ่งระดับ เป็นเงาด้านหลังของห้องเดียวกัน
- **Highlight** (`#161D2E`) — แสงที่ส่องจากด้านหลังกระเบื้อง ใช้สำหรับ cursorline และ selection
- **Errors** ใช้ Coral Red (`#D45C5C`) — สีแดงอบอุ่นที่ส่งสัญญาณเตือนโดยไม่ทำให้วิตกกังวล

### 4. Cursor เป็นจุดยึด

Cursor ใช้สี Ember (`#E8721A`) เหตุผลเดียวกับที่จอซ้ายในภาพต้นแบบเป็นจุดที่ร้อนที่สุดในห้องที่เย็นสบาย — มันเป็นจุดที่สายตากลับมาหาเสมอ Cursor สีส้มลุกโชนนี้ไม่สับสนกับชื่อฟังก์ชัน (เพราะมันเคลื่อนที่) และไม่สับสนกับ keyword (เพราะ keyword ใช้สี amber ไม่ใช่ orange)

## การจับคู่ฟอนต์

ธีม "Cinder Grid" เข้ากันได้ดีกับ:

- **หลัก:** JetBrains Mono — รูปแบบตัวอักษรเรขาคณิตของมันเหมือนกับกริดกระเบื้อง: โมดูลาร์ หนาแน่น แต่ละตัวอักษรอยู่ในเซลล์ที่แม่นยำ ฟังก์ชันสีส้มจะเรืองแสงสดใสยิ่งขึ้นเมื่อใช้กับฟอนต์นี้
- **ทางเลือก:** Fira Code — รูปทรงที่ไหลลื่นกว่าเล็กน้อยช่วยให้ชุดสีอุณหภูมิสูงดูนุ่มนวลขึ้น

Ligatures (การรวมอักษร) เช่น `=>`, `!==`, `>=` ช่วยลดความยุ่งเหยิงทางสายตาของ operators และทำให้ keywords สีอำพันและฟังก์ชันสีส้มเป็นจุดสำคัญบนบรรทัดต่อไป

## ความรู้สึกเมื่อใช้งานจริง

ลองนึกภาพการเปิดไฟล์ Rust ตอนตีสองพร้อมกับวิวเมืองนอกหน้าต่าง: พื้นหลัง (`#0B0F1A`) วางตัวเหมือนห้องที่คุณอยู่มานานจนไม่สังเกตเห็นผนังแล้ว สายตาของคุณไปที่การประกาศ `fn` ครั้งแรกทันที — สีส้มลุกโชน (`#E8721A`) ที่ร้อนที่สุดบนหน้าจอ เมื่ออ่านลงมาในเนื้อหาฟังก์ชัน keywords สีอำพัน `#D4A827` ทำหน้าที่เครื่องหมายจุดตัดสินใจ — `if`, `match`, `return` — เหมือนคานที่คุณต้องรู้ตำแหน่งก่อนเดินไปในห้อง การประกาศประเภทข้อมูลปรากฏในสีน้ำเงินกระเบื้อง `#4D8FCC` เย็นกว่าและรับน้ำหนักโครงสร้าง String literals อ่านในสี Moss `#5FAD6E` ความนุ่มนวลเพียงอย่างเดียวในชุดสี ตัวเลขอยู่ใน Gold `#C8901A` ตายตัวและเป็นวัสดุ ความคิดเห็นแทบมองไม่เห็น ถอยอยู่ในช่องว่างระหว่างกระเบื้อง และตลอดทั้งหมดนั้น จุด ember เพียงหนึ่งจุด — cursor ที่ `#E8721A` — เคลื่อนไหวพร้อมกับคุณ เป็นด้ายลุกโชนเพียงเส้นเดียวที่คุณติดตามผ่านกริด

นั่นคือ Cinder Grid.
