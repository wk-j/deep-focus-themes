# Frozen Sprawl

A dark coding color scheme using a **deep slate-blue base** with a **glacial spectrum** — `#78C8E0` glacial cyan functions, `#70B890` melt-green strings, `#5890B8` steel-blue types, `#E0A840` amber constants. Built for cold-focused, high-concentration work sessions.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#0E1520` | Slate Abyss |
| Sidebar / Panels | `#080E18` | Deep Ice |
| Keywords | `#E8F4FF` | Frost White |
| Functions | `#78C8E0` | Glacial Cyan |
| Strings | `#70B890` | Melt Green |
| Types & Tags | `#5890B8` | Steel Blue |
| Numbers | `#E0A840` | Amber Core |
| Constants & Booleans | `#60B8A8` | Ice Teal |
| Variables | `#D8E8F4` | Ice Mist |
| Comments | `#4E6880` | Frozen Depth |
| Attributes | `#8AAAC8` | Spire Slate |
| Errors | `#C85050` | Thaw Red |

## Key Ideas

- **Ice-blue spectrum.** The entire palette derives from the blue-white-teal range of a frozen cityscape — no warm colors except the cursor anchor.
- **Glacial cyan as hero.** Functions use `#78C8E0`, the color of sunlight diffusing through an ice sheet — the most eye-catching element in a frozen world.
- **Frost white keywords.** Structural keywords appear in near-white `#E8F4FF`, the color of ice crystals at the surface — always visible, never competing.
- **Depth over borders.** Panels separated by shade, not lines.
- **Amber cursor.** The `#E0A840` caret is the single warm point — the one element that doesn't belong to the ice, and therefore impossible to miss.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
