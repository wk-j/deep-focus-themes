# Frozen Sprawl — แนวคิดโทนสี

## ที่มา

ภาพอ้างอิงคือมุมมองจากอากาศของเมืองใหญ่ที่ถูกปกคลุมด้วยน้ำแข็ง ตึกระฟ้าและอาคารนับพันหลังถูกน้ำแข็งครอบงำ ไม่ใช่แค่ผิวเคลือบ แต่แข็งทั้งหมด ทางด้านขวาของภาพคือมหาสมุทรที่แข็งตัวสนิท ผิวหน้าเรียบและสว่างขาว ท้องฟ้าเหนือเมืองเป็นสีขาว-ฟ้าอ่อนที่กระจายแสงอย่างสม่ำเสมอ ไม่มีทิศทางแน่ชัด สีที่อบอุ่นเพียงอย่างเดียวในภาพคือขอบอาคารด้านล่างซ้ายที่รับแสงอาทิตย์มุมต่ำ ให้สีแอมเบอร์เล็กน้อยในโลกที่เป็นน้ำแข็งทั้งหมด

---

## แนวคิดหลัก

Frozen Sprawl เป็นโทนสีสำหรับเขียนโค้ดที่ได้แรงบันดาลใจจากเมืองที่ถูกธารน้ำแข็งกลืน ความตึงเครียดหลักของโทนสีนี้คือ ทุกอย่างเย็นและเป็นสีฟ้า-ขาว ยกเว้น cursor สีแอมเบอร์ที่เป็นจุดอบอุ่นเดียวในทั้งโทนสี

การออกแบบแก้ปัญหานี้ด้วยสามการตัดสินใจ:

- **พื้นหลังลึกสีฟ้า** — พื้นหลัง (`#0E1520`, `#080E18`) เป็นสีน้ำเงินเข้ม คล้ายเงาของตึกล่างที่ท้องฟ้าสีขาวส่องลงมา ไม่ใช่สีดำทึบ แต่มีสีฟ้าผสมอยู่เสมอ
- **น้ำแข็งหลายเฉด** — accent แต่ละสีไม่ใช่ฟ้าเหมือนกัน Glacial Cyan (`#78C8E0`) คือแสงแดดผ่านน้ำแข็ง Steel Blue (`#5890B8`) คือผิวตึกในแสงเมฆ Melt Green (`#70B890`) คือสีสิ่งมีชีวิตที่รอดจากความหนาวที่ขอบน้ำแข็ง
- **แอมเบอร์เป็นข้อยกเว้น** — cursor สี `#E0A840` อบอุ่นเพราะไม่มีอะไรอบอุ่นอีกแล้ว ทำให้หาตำแหน่ง cursor ได้ทันทีในทุกสถานการณ์

## หลักการออกแบบ

### 1. ทุกสีเย็น ยกเว้นหนึ่งจุด

โทนสีใช้ช่วงสีฟ้า-ขาว-เขียวน้ำแข็งจากภาพ และเพิ่มสีอบอุ่นเพียงหนึ่งสี Glacial Cyan (`#78C8E0`) คือสีหลัก (hero color) เลือกเพราะอ่านว่า "เย็นและโปร่งใส" มากกว่า aqua ทั่วไป Steel Blue (`#5890B8`) อ้างอิงจากผิวคอนกรีตของตึกในแสงสว่างจาง Ice Teal (`#60B8A8`) อยู่ในช่วงระหว่างฟ้าและเขียวที่บอกถึงน้ำแข็งละลาย

### 2. ลำดับชั้นของ Syntax

| Role | สี | Hex | เหตุผล |
|---|---|---|---|
| Keywords | Frost White | `#E8F4FF` | ผิวน้ำแข็งที่รับแสงมากสุด — โครงสร้างหลัก มองเห็นง่าย |
| Functions | Glacial Cyan | `#78C8E0` | Hero color — แสงแดดผ่านน้ำแข็ง สว่างที่สุดในโทนสี |
| Strings | Melt Green | `#70B890` | สีสิ่งมีชีวิตที่รอดรอดชีวิต แตกต่างจากสเปกตรัมฟ้า |
| Types | Steel Blue | `#5890B8` | ผิวตึกในแสงเมฆ — โครงสร้าง กลาง เชื่อถือได้ |
| Attributes | Spire Slate | `#8AAAC8` | ผิวตึกบนสูงที่รับแสงท้องฟ้าจาง |
| Constants | Ice Teal | `#60B8A8` | สีน้ำแข็งละลาย — ระหว่างฟ้าและเขียว นิ่ง |
| Numbers | Amber Core | `#E0A840` | ความอบอุ่นของแสงขอบตึก — โดดเด่นจากสีเย็น |
| Comments | Frozen Depth | `#4E6880` | ชั้นล่างของตึกที่ไม่มีแสงส่อง — มัว แต่อ่านได้ |

### 3. พื้นหลังเป็นฉาก ไม่ใช่ความว่างเปล่า

พื้นหลังมีสีฟ้าผสมอยู่เสมอ ไม่ใช่สีดำกลาง:

- **พื้นที่ editor** (`#0E1520`) — สีน้ำเงินเข้มของเงาชั้นล่างตึก มืดแต่ยังมีสีฟ้า
- **แผงด้านข้าง (sidebar)** (`#080E18`) — น้ำเงินเข้มกว่า เหมือนพื้นที่ระหว่างตึกที่แสงไม่ถึง ดูถอยหลังไปในเชิงลึก
- **Highlight** (`#1A2535`) — โซนเงากลางที่แสงท้องฟ้าส่องเฉียงมาพอดี ทำให้การเลือก selection หรือ hover ดูเหมือนประกายแสงอ่อน
- **Errors** ใช้ Thaw Red (`#C85050`) — สีแดงที่ desaturated แล้ว อ่านว่า "สัญญาณ" โดยไม่ทำให้รู้สึกตื่นตระหนก

### 4. Cursor เป็นจุดยึด

Cursor ใช้ Amber Core (`#E0A840`) ในภาพอ้างอิง สีแอมเบอร์ปรากฏเฉพาะบริเวณขอบอาคารด้านล่างซ้ายที่รับแสงอาทิตย์มุมต่ำ นั่นคือจุดเดียวในภาพที่มีความอบอุ่น ใน editor cursor สีแอมเบอร์ทำหน้าที่เดียวกัน: หาตำแหน่งได้ทันที ไม่ว่าจะอยู่บนพื้นหลังหรือสีข้อความใด

## การจับคู่กับ Font

โทนสี Frozen Sprawl เหมาะกับ:

- **หลัก:** JetBrains Mono — ความแม่นยำทางเรขาคณิตสะท้อนกริดเมืองที่มองจากอากาศ ทุกมุมเป็น 90 องศา ทุกพื้นผิวเรียบภายใต้น้ำแข็ง
- **ทางเลือก:** Fira Code — contrast ของเส้นที่อ่อนกว่าสอดคล้องกับพื้นผิวมหาสมุทรแข็งตัวในภาพ ซึ่งดูเรียบจากระยะไกลแต่มีลายละเอียดเมื่อมองใกล้

Ligature เช่น `=>`, `!==`, `>=` ในโทนนี้อ่านว่าความแม่นยำทางกลไกในสภาพแวดล้อมที่ต้องการความชัดเจนสูง

## ความรู้สึกเมื่อใช้งานจริง

ลองนึกภาพเปิดไฟล์ configuration ระบบในชั่วโมงก่อนรุ่งสาง: พื้นหลัง editor แสดงเป็นสีน้ำเงินเข้ม (`#0E1520`) และสิ่งแรกที่ตาพบคือ cursor สีแอมเบอร์ (`#E0A840`) — จุดอบอุ่นเดียวในทั้งหน้าจอ Keywords ปรากฏเป็นสีขาวน้ำแข็ง (`#E8F4FF`) Functions มาในสีฟ้าน้ำแข็ง (`#78C8E0`) — สีที่สว่างและโดดเด่นที่สุดในโทนสี Strings แสดงเป็นสีเขียวอ่อน (`#70B890`) แตกต่างจากสเปกตรัมฟ้าอย่างชัดเจน Types ถือสีน้ำเงินเหล็ก (`#5890B8`) นิ่งและเชื่อถือได้ Comments ถอยหลังเป็นสีเข้มจาง (`#4E6880`) มองเห็นได้แต่ไม่รบกวน แผง sidebar นั่งอยู่ในสีน้ำเงินเข้มกว่า (`#080E18`) และตาอ่านว่าอยู่ห่างออกไป ตลอดเซสชันยาว โทนสีนี้ไม่ล้า เพราะสีเย็นไม่ล้า และเมื่อต้องการหาตำแหน่ง cursor สีแอมเบอร์อยู่ที่นั่นเสมอ อบอุ่น และไม่เคยแข็งตัว

นั่นคือ Frozen Sprawl.
