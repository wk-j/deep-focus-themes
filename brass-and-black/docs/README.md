# Brass and Black

A dark coding color scheme using a **deep charcoal base** with a **cinematic brass-and-jewel spectrum** — `#C8B800` brass functions, `#8CB87A` sage strings, `#4E9ED4` blue types, `#9A88CC` lavender constructors. Built for high-focus, late-night sessions where every symbol earns its color.

## Palette

| Token | Hex | Name |
|---|---|---|
| Background | `#181818` | Stage Black |
| Sidebar / Panels | `#111111` | Pit Black |
| Keywords | `#F0EAD8` | Parchment White |
| Functions | `#C8B800` | Brass |
| Strings | `#8CB87A` | Sage |
| Types & Tags | `#4E9ED4` | Steel Blue |
| Numbers | `#3AADA8` | Teal |
| Constants & Booleans | `#E8D000` | Gold |
| Variables | `#E8E0CC` | Warm Ivory |
| Comments | `#665C48` | Scorched Oak |
| Attributes | `#7ABCE8` | Sky |
| Errors | `#E8363A` | Signal Red |

## Key Ideas

- **Brass as hero.** The acid-brass `#C8B800` — lifted directly from the image's tab highlight — makes every function call feel like a downbeat.
- **Parchment keywords.** Structural flow markers (if, for, return) are bold near-white, standing apart without competing with accent colors.
- **Jewel-toned spectrum.** Six distinct hues span the syntax tree: brass → sage → blue → teal → lavender → amber. No two adjacent token types share a hue family.
- **Depth over borders.** Panels separated by shade (`#181818` editor, `#111111` sidebar), not lines.
- **Gold cursor.** The `#E8D000` caret is unmissable against the dark stage — always center stage.
- **Pair with:** JetBrains Mono or Fira Code (ligature support).
